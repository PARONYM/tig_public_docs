# アイテム詳細の表示（モーダル）
{: .no_toc }

アイテムをタップしたときに表示されるモーダル（ポップアップ）の概要です。

1. TOC
{:toc}

## モーダルの種類

| 種類 | 内容 |
|------|------|
| Webページ表示 | 商品ページなどを、視聴を中断せずポップアップ内で表示 |
| 記事 | 画像・タイトル・本文（リンク付き）を表示 |
| 画像カルーセル | 複数画像をスワイプで切り替え表示 |
| クーポン | クーポンコードを表示し、コピーできる |
| アンケート・投票 | 選択肢に回答でき、結果を割合で表示 |
| 商品（EC連携） | 商品画像・バリエーション・数量を選んでカートに追加 |
| チャットボット | チャットで問い合わせ・案内 |
| ヒント表示 | 並び替えなどの企画コンテンツのヒントを表示 |

モーダルは閉じるボタンで閉じられ、表示中にアイテムをストックすることもできます。

## チャットボットについて

チャットボットは他のモーダルと異なり、**動画を停止せずに**利用できます。
