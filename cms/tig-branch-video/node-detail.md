# Tig Branch Video 個別動画の詳細
{: .no_toc }

Tig Branch Video（分岐動画）を構成する**1本ずつの動画**の設定を行う画面です。コンテンツ詳細の「動画設定」タブの一覧から、各動画を開いて編集します。

## 画面の構成

この画面は大きく2つの領域に分かれています。

- **ヘッダー部分（画面上部）**: パンくず（Tig Branch Video → コンテンツ名 → 動画名）、サムネイル、動画の変更・プレビュー、状態表示（変換中・変換エラー）。
- **メイン領域（画面下部）**: タブ（基本設定・オプション設定・分岐動画設定・アイテム設定）を切り替えて編集します。**オプション設定**で「動画が終わったときの動作」を、**分岐動画設定**で「次につながる動画（分岐先）」を設定します。

以降は、各タブでできることを説明します。

## メイン領域での操作（タブ）

| タブ | 内容 |
|------|------|
| 基本設定 | タイトル・説明・所有グループ・備考、サムネイルの変更 |
| オプション設定 | シークバーの表示、**終了時の動作**（停止／ループ／別の動画へ自動遷移／ストック一覧表示）、遷移先の指定 |
| 分岐動画設定 | この動画から**分岐先**（次に再生できる動画）を追加・解除 |
| アイテム設定 | 動画に表示するアイテム（ボタン・リンク）の管理 |

## 終了時の動作

「オプション設定」で、動画が最後まで再生されたときの動作を選べます。

- **停止**: その場で停止します。
- **ループ**: 先頭から繰り返します。
- **別の動画へ自動遷移**: 指定した動画へ自動でつなぎます（遷移先と開始位置を指定）。
- **ストック一覧表示**: ストックした一覧を表示します。

## 分岐先の追加・解除

「分岐動画設定」タブから、この動画の**分岐先**（視聴者が選んで進める次の動画）を設定できます。

1. 「登録動画から追加」で、同じコンテンツ内の動画から分岐先を選びます。
2. 分岐先ごとにタイトル・開始位置などを設定して登録します。
3. 不要になった分岐先は「解除」で外せます。

## 削除について

- 動画は削除できますが、**最初に再生される導入動画**や、**ホームボタンの遷移先**に設定されている動画は削除できません。
- 削除した動画を遷移先にしていた他の動画は、終了時の動作が自動的に「停止」に変わります。
