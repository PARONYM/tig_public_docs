## 公開制御・アクセス制限

配信ごとに、視聴できる相手を制限できます。

| 制限 | 内容 |
|------|------|
| 視聴パスワード | パスワードの入力を求め、一致した場合のみ視聴できます |
| IPアドレス制限 | 許可した接続元からのみ視聴できます |
| ドメイン制限 | 許可したサイトに設置した場合のみ視聴できます |
| 非公開プレビュー | 公開前の確認用に、専用パスワードでの視聴を許可できます（アーカイブの確認用） |

## 表示言語

視聴者のブラウザの言語設定に応じて、**日本語・英語**の表示を自動で切り替えます。対応していない言語の場合は英語で表示されます。

## 配信者ブランドの配色

配信ごとに、プレイヤー各所の配色を設定できます。

| 設定できる箇所 | 反映先 |
|------|------|
| メインカラー | コメント投稿ボタン・紹介アイテムの詳細ボタン |
| Tigアイコンの色 | 動画上のアイテムアイコン・文字色 |
| 公式マーク・公式コメントの色 | スタッフ発言の見た目 |
| ストック数バッジの色 | お気に入り件数の表示 |

配信後の画面（フィラー）では、一部の配色（メインカラー・ストックバッジ）のみが反映されます。
