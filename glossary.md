# 用語集

本サイトで使う用語をまとめています。各機能ページの「用語」欄は、この用語集からの抜粋です。
用語の追加・更新は、原則このページ（用語集）を正として行います。

**「対象」** 欄は、その用語が主にどの製品・領域で使われるかを示します（`共通` は製品をまたいで使う語）。

| 用語 | 意味 | 対象 |
|------|------|------|
| UU（ユニークユーザー） | 重複を除いた利用者の数 | 共通 |
| 視聴 | 動画の再生が開始されたこと | 共通 |
| 総再生数 | 再生が開始された延べ回数（同じ人による複数回もそれぞれ数える） | LIVE |
| 5秒以上再生数 | 5秒以上再生された延べ回数 | LIVE |
| 視聴UU | 再生した人数（重複を除いたユニーク数） | LIVE |
| 新規視聴UU | 過去6ヶ月、同じグループのライブを一度も視聴していない新規の視聴者数 | LIVE |
| ピーク視聴UU | 配信中の同時視聴者数の最大値 | LIVE |
| ライブ終了後UU | ライブ終了後の画面（事後フィラー）を訪れた人数 | LIVE |
| 再生UU | 再生した人数（重複を除いたユニーク数。アーカイブ配信では「視聴UU」をこの表記にしている） | LIVE |
| 再生完了率 | 動画のどこまで再生されたか。25% / 50% / 75% / 100% の各地点まで再生した割合 | LIVE |
| 平均再生率 | 平均して動画のどれくらいの割合まで再生されたか | LIVE |
| コメント | ライブ配信中に視聴者が投稿するコメント | LIVE |
| Like | 視聴者が送る「いいね」 | LIVE |
| ストック | アイテムを後で見るために保存すること | 共通 |
| ジャンプ | アイテムのリンク先URLへ遷移すること | 共通 |
| ダイレクトジャンプ | アイテムをタップした際にリンク先へ直接遷移する挙動 | 共通 |
| マルチリンク | 1つのアイテムに複数のリンクを設定する機能 | 共通 |
| 視聴行動UU合計 | ストック・ジャンプのいずれかを1回以上行った人数 | LIVE |
| タップ | 映像領域をタップ（タッチ／クリック）する操作 | 共通 |
| Tig領域 | 映像上で、アイテム等の Tig の UI が表示される領域 | 共通 |
| Tigアイコン（吹き出し） | 動画上に表示される、アイテムへの導線となるアイコン・吹き出し | 共通 |
| リアルタイム推移 | ライブ配信中の各指標（視聴者・コメント・Like など）の時系列の変化 | LIVE |
| 計測タグ（ta_tag） | 流入元を区別するためにURLへ付与する計測用のタグ | LIVE |
| 紹介アイテム数 | ライブ中に紹介したアイテムの数 | LIVE |
| 内部プレビュー | 公開前・未承認の状態でも内容を確認できるプレビュー | CMS |
| ライブ配信 | リアルタイムで配信される動画。配信中の指標を計測できる | LIVE |
| アーカイブ配信 | ライブ配信の終了後に、録画として視聴できる動画 | LIVE |
| アーカイブ取込（外部アーカイブ） | 外部で配信・録画された動画をアーカイブとして取り込むこと | LIVE |
| フィラー | ライブの各場面（事前・オープニング・ブレイク・エンド・事後）で表示される待機・つなぎ画面 | LIVE |
| 事前フィラー | ライブ開始前に表示される待機画面 | LIVE |
| 事後フィラー | ライブ終了後に表示される画面 | LIVE |
| 配信者メニュー | ライブ中に表示される、配信者の情報・リンクのメニュー | LIVE |
| スタッフモード | 配信関係者が確認用に視聴するモード（専用URL・パスワードで利用） | LIVE |
| サイマル配信（シミュキャスト） | Tig LIVE の配信を YouTube / Instagram / TikTok / X などの外部サービスへ同時配信すること | LIVE |
| FHD配信 | フルHD（高解像度）での配信 | LIVE |
| プレミア公開 | 事前にアップロードした動画を、指定した時刻に自動でライブ開始する公開方法 | LIVE |
| 公開モード | コンテンツの公開状態（公開／非公開／期間指定） | CMS |
| 承認フロー | コンテンツを公開する前に承認／未承認を管理する仕組み（グループ単位で有効化） | CMS |
| 配信テンプレート | ライブ配信の新規作成時に、各種設定をまとめて引き継ぐためのテンプレート | LIVE |
| インサイト | Tig LIVE の配信終了後の視聴指標を、複数配信まとめて確認できる画面（サイドバーのタブ） | LIVE |
| ヒートマップ | 動画のどこがよくタップされたか（タッチ座標）を動画に重ねて可視化する分析 | 共通 |
| 埋め込みコード | サイトにプレーヤーを設置するためのコード（ダイレクト／モーダル／フローティングの形式がある） | CMS |
| OGP画像 | SNS等で共有された際に表示される画像 | CMS |
| アニメーションサムネイル | サムネイルにマウスを乗せた際に再生される短いアニメーション | Short |
| コレクション | コンテンツをまとめるグループ（一覧の絞り込みなどに使う） | Short |
| UGC | 利用者が作成・投稿したコンテンツ（Instagram / TikTok 等から取り込む動画など） | Short |
| グループ | コンテンツをまとめる単位。新規視聴UU の判定範囲などに使われる | 共通 |
| 親グループ | 子グループを束ねる上位のグループ。配下グループの管理元（承認依頼メールの宛先などに使われる） | 共通 |
| アイテムマスター | コンテンツに紐づかない、グループ共通の商品アイテム（マスタ）。商品アイテム管理で扱う | CMS・共通 |
| ユニークキー | 商品アイテムを識別するためのキー（商品識別子） | CMS |
| EC商品コード | EC連携で用いる商品コード | CMS |
| Tig Short | 短尺の縦型動画に対応した Tig プラットフォームの製品 | Short |
