# コメント・いいね
{: .no_toc }

LIVE配信の視聴中に使えるコメント・いいねなどの参加機能の概要です。

## 目次
{: .no_toc .text-delta }

1. TOC

## コメント

- はじめて投稿するときにニックネームを設定します（アカウント登録は不要です）。
- 投稿したコメントは、数秒以内に視聴者全員の画面に流れます。
- 運営側で設定した NGワードを含むコメントは、他の視聴者には表示されません。
- 配信スタッフのコメントには公式バッジが付き、大切なお知らせは画面上部にピン留め表示されます。

## いいね

- ハートボタンから、種類を選んでいいねを送れます（時期によって季節限定のアイコンが登場します）。
- 自分のいいねはすぐに画面にアニメーション表示され、他の視聴者のいいねも流れてきます。

## 画面の表示

| 機能 | 内容 |
|------|------|
| 視聴者数 | 現在の視聴者数がヘッダに表示される |
| コメント欄の開閉 | パソコンではコメント欄を開閉、スマートフォンでは表示/非表示を切り替えられる |
| スタッフのピン留め | 重要なコメントが画面上部に固定表示される |

## アーカイブでのコメント

チャット付きのアーカイブ（録画）では、配信中に投稿されたコメントやピン留めが、当時と同じタイミングで再現されます。
