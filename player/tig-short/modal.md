# 商品モーダル（ポップアップ表示）
{: .no_toc }

商品ポイントをタップしたときに、プレイヤー内に開く商品情報のポップアップ（モーダル）についての説明です。

{: .no_toc .text-delta }

1. TOC

## モーダルとは

商品ポイント（Tigポイント）をタップしたとき、ページを移動せずに**プレイヤー内のポップアップ**で商品情報を表示する方法です。表示のしかたはコンテンツの設定で選べます。

## 表示される内容

| タイプ | 内容 |
|--------|------|
| ページ表示（iframe） | 商品ページなどのURLをポップアップ内に表示します |
| Shopify商品 | Shopify と連携している場合、商品情報・カートをポップアップで表示します |
| プレイリスト | プレイリストのアイテムをポップアップで表示します |

## 大きさについて

ポップアップの大きさは、コンテンツの設定に応じて「小さめ」「中くらい」「全画面」で表示されます。

## 再生との関係

- モーダルを開くと、再生中の動画はいったん**停止**します。
- モーダルを閉じると、動画の再生が**再開**します（開く前から停止していた場合は再開しません）。

## ストックボタン

設定によっては、モーダル内に**ストック（あとで見る）ボタン**が表示され、その場で商品を保存できます。
