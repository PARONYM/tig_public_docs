# Tig Branch Video アナリティクス
{: .no_toc }

Tig Branch Video（分岐動画）の視聴状況を分析する画面です。再生数やタップなどの基本指標に加え、**分岐ごとの流入・離脱**を確認できます。

## 画面の構成

- **ヘッダー部分（画面上部）**: 期間の指定（全期間／期間指定・プリセット）、表示ボタン、CSVダウンロード。
- **メイン領域（画面下部）**: 期間内の集計結果（再生数・再生デバイス・計測タグ別・タップ・ストック・URL遷移・分岐一覧）。管理者は**分岐経路**も確認できます。

以降は、表示される内容を説明します。

## 表示される指標

| 指標 | 内容 |
|------|------|
| 再生数 | 総再生数・視聴UU（重複を除いた人数）。読み込み回数は管理者向け表示 |
| 再生デバイス | 視聴された OS の内訳（iOS／Android／その他） |
| 計測タグ別再生数 | 視聴URLに付けた[計測タグ（ta_tag）]({{ '/glossary.html' | relative_url }})ごとの再生数 |
| タップ | 映像へのタップ数（Tig領域／非Tig領域の内訳） |
| ストック・URL遷移 | アイテムのストック数・ジャンプ（リンク遷移）数（総数とユニーク） |
| 分岐一覧 | 分岐元ごとの再生数・分岐先への分岐数・**[離脱数]({{ '/glossary.html' | relative_url }})**（分岐先へ進まずに離脱した数） |

> 割合(%)の分母など細かな数え方は[用語集]({{ '/glossary.html' | relative_url }})も参照してください。

## 分岐一覧

分岐動画では、どの動画（分岐元）からどの動画（分岐先）へ、どれだけ視聴者が進んだかを一覧で確認できます。分岐先へ進まずに離脱した数（離脱数）もわかるため、どの分岐で視聴者が離れているかを把握できます。

## 分岐経路（管理者向け）

管理者は、視聴者が **導入動画 → 次の動画 → … とどの経路をたどったか**を、分岐の階層ごとに集計した「[分岐経路]({{ '/glossary.html' | relative_url }})」を確認できます。各経路の到達ユニーク数・URL遷移数・遷移率がわかります。

## CSVダウンロード

集計のもとになるイベント（再生・タップ・ストック・分岐遷移など）を CSV で出力できます。
