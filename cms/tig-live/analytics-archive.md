# Tig LIVE アナリティクス（アーカイブ配信）
{: .no_toc }

アナリティクス画面の **アーカイブタブ**で見られる項目です。配信終了後（アーカイブ）の視聴状況を、**期間を指定して**振り返れます。LIVE配信タブは別ページ「[アナリティクス（LIVE配信）]({{ '/cms/tig-live/analytics-summary.html' | relative_url }})」を参照してください。

## 画面の構成

アーカイブ配信の各指標を上から順に確認できる画面です（画面上部の LIVE／アーカイブ／シーン分析 の切り替えで「アーカイブ」を選択）。**上部にグラフ（推移）**、その下に 再生数サマリー・タップ分析・再生完了率・計測タグ別・アイテム別・ユーザー行動別 などの指標が並びます。

以降は、各セクションの項目を説明します。

## 使い方

1. CMS で対象のライブコンテンツを開く
2. 「アナリティクス」→ 上部のタブで **「アーカイブ」** を選ぶ（アーカイブが作成されている配信のみ）
3. 必要に応じて**期間を指定**して集計を絞り込む

> アーカイブタブは LIVE配信タブと多くの項目が共通です（用語は[用語集]({{ '/glossary.html' | relative_url }})を参照）。LIVE配信中だけの項目（リアルタイムの推移グラフ・同時視聴のピーク・コメント/いいねの総数など）は、アーカイブには表示されません。

## 1. グラフ（推移）

アーカイブの推移を折れ線グラフで表示し、ボタンで系列を切り替えられます。

| 系列 | 内容 |
|------|------|
| 再生完了率 | 動画のどこまで再生されたかの推移（既定表示） |
| タップ / ストック / ジャンプ | 各操作の推移 |

## 2. 再生数サマリー

| 項目 | 意味 |
|------|------|
| 総再生数 | 再生が開始された延べ回数 |
| 5秒以上再生数 | 5秒以上再生された延べ回数 |
| 再生UU | 再生した人数（重複を除いたユニーク数） |
| 5秒以上再生UU | 5秒以上再生した人数 |

## 3. ストック・ジャンプ（総数）

| 項目 | 意味 |
|------|------|
| ストック | アイテムをストックした延べ人数 |
| ジャンプ | リンク遷移（ジャンプ）の総数 |

> 総数と「再生UUに対する割合(%)」を表示します。

## 4. タップ分析

| 項目 | 意味 |
|------|------|
| 総タップ回数 | 映像領域へのタップ総数（再生1回あたりの平均も表示） |
| Tig領域タップ / 非Tig領域タップ | タップの内訳 |

## 5. 再生完了率（アーカイブ配信のみ）

動画が**どこまで再生されたか**を示します。

| 項目 | 意味 |
|------|------|
| 25% / 50% / 75% / 100% 再生 | 動画の各地点まで再生した割合 |
| 平均再生率 | 平均してどれくらいの割合まで再生されたか |

## 6. 計測タグ別（ta_tag）

視聴URLに付けた**計測タグ**ごとに、視聴UU・ストック・ジャンプを確認できます。

## 7. アイテム別

| 項目 | 意味 |
|------|------|
| 紹介アイテム数 | 動画中に紹介したアイテム数 |
| アイテム別 ストック / ジャンプ | アイテムごとの数 |
| マルチリンク ジャンプ | マルチリンクの子リンクごとのジャンプ数 |

## 8. ユーザー行動別

「人数（UU）」と「再生UUに対する割合(%)」で表示します。

| 項目 | 意味 |
|------|------|
| ストックアクション（UU） | 1回以上ストックをした人数 |
| ジャンプアクション（UU） | 1回以上ジャンプをした人数 |
| 視聴行動UU合計 | ストック・ジャンプのいずれかを1回以上行った人数 |

> このほか、お知らせ・過去アーカイブ誘導・クーポン取得・配信者メニュー表示などの行動人数も表示されます。

## 9. シーン分析

「シーン分析」ボタンから、動画のシーン（時間帯）ごとの推移を、動画を見ながら確認できます。

## CSVダウンロード

アーカイブのイベントログ・滞在時間などを、**指定した期間で**CSVダウンロードできます。

## 補足

- 「UU」は端末（ブラウザ）単位で集計されます。
- 割合(%)は原則「**再生UU**」を分母にします。
- LIVE配信中だけの項目（リアルタイム推移・ピーク視聴・コメント/いいね総数など）はアーカイブには表示されません。
