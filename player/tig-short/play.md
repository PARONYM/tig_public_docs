# 動画の再生・操作
{: .no_toc }

Tig Short プレイヤーで、動画を再生・停止したり、音量やシーク（頭出し）、商品ポイント間の移動を行うための操作の説明です。

## 操作の一覧

| 操作 | 内容 |
|------|------|
| 再生 / 停止 | 再生ボタンで再生、もう一度押すと停止します。ボタンのアイコンは今の状態に合わせて切り替わります |
| 音量（ミュート解除） | 動画は最初は音が消えた状態で再生されます。音量ボタン（またはミュート解除の案内）をタップすると音が出ます |
| シーク（頭出し） | 画面下部のバーをドラッグすると、見たい位置へ移動できます。ドラッグ中はプレビュー画像と時間が表示されます |
| 前 / 次の商品ポイントへ移動 | 動画内の商品ポイント（Tigポイント）へ、前後にスキップできます。先頭・末尾ではボタンが押せなくなります |
| 動画の切り替え | 上下スワイプで次／前の動画に切り替わります（設定によってはアニメーションで切り替わります） |

## 音について

スマートフォンのブラウザは、最初から音を出す自動再生を制限しています。そのため動画は**最初はミュート（消音）で再生**され、画面の案内やボタンから**ミュート解除**すると音が出ます。

## シークバーの見かた

- バー上の点は、動画内に**商品ポイント（Tigポイント）がある位置**を示します。
- バーをドラッグしている間は、その位置の**サムネイル画像と時間**が表示されます。

## 表示される操作ボタンについて

再生ボタン・音量ボタン・シークバー・スキップボタン・ストックボタンは、**コンテンツの設定によって非表示にできます**。表示されていない場合は、その動画でその操作が無効になっています。
