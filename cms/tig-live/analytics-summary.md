# Tig LIVE アナリティクス サマリー数値

Tig LIVE の管理画面「アナリティクス」→ **LIVE配信タブ**の最上部に並ぶ数値サマリーです。
配信1本がどれだけ届いたか（再生回数・視聴人数・新規の割合）を把握できます。

## 使い方

1. CMS で対象のライブコンテンツを開く
2. 「アナリティクス」→「LIVE配信」タブを選ぶ
3. 画面上部のサマリー数値とグラフで到達状況を確認する

## 用語

| 用語 | 意味 |
|------|------|
| UU | ユニークユーザー（重複を除いた人数） |
| 視聴 | 動画の再生が開始されたこと |

## 仕様項目

| 項目 | 仕様 |
|------|------|
| 総再生数 | ライブ配信中に再生が開始された延べ回数 |
| 5秒以上再生数 | ライブ配信中に5秒以上再生された延べ回数 |
| 視聴UU | 再生した人数（重複を除いたユニーク数） |
| 新規視聴UU | 過去6ヶ月、同じグループのライブを一度も視聴していない新規視聴者数 |
| リアルタイム推移（視聴者／コメント／Like 等） | ライブ配信中の各指標の時系列推移をグラフで表示 |

### 補足

- 「視聴UU」「新規視聴UU」は端末（ブラウザ）単位で集計されます。同じ方でも端末や環境が変わると別の視聴者として数えられる場合があります。
