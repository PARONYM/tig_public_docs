# 出演者・パフォーマー
{: .no_toc }

ライブなどに登場する「配信者（出演者・パフォーマー）」を管理し、コンテンツに紐付けます。専用画面ではなく、コンテンツ編集の「配信者」から追加・選択します。

## 配信者マスタ

配信者は、次の情報を持つ「マスタ」としてグループ単位で管理します。一度登録すれば、複数のコンテンツで使い回せます。

| 項目 | 内容 |
|------|------|
| 名前 | 配信者の表示名 |
| サムネイル画像 | 配信者のアイコン画像（新規登録時に設定。以降は上書きのみ） |
| 説明 | 配信者の紹介文 |
| リンク | Instagram・X・TikTok・ECサイト・メールマガジン、およびその他リンク |

## できること

| 操作 | 内容 |
|------|------|
| 一覧 | 登録済みの配信者を一覧表示 |
| 追加・編集 | 配信者マスタを新規作成・更新 |
| コンテンツへ紐付け／解除 | コンテンツ（や配信テンプレート）に配信者を関連付け／解除 |

## 親グループの配信者の共有

設定（親グループ参照）が有効なグループでは、**親グループに登録された配信者を選んで使う**こともできます。親グループ由来の配信者は、自グループからは選択・表示のみで、編集はできません。
