# EC連携（商品購入・カート）
{: .no_toc }

動画内の商品をタップして、商品情報の確認・バリエーション/数量の選択・カート追加ができる EC 連携機能の説明です。

{:toc}

## 対応しているECサービス

Shopify・Makeshop・ショップサーブ の商品を、プレイヤー内の商品モーダルで表示・購入操作できます。楽天は専用のランディングページで対応します。その他のサービスは、商品ページをそのままポップアップ（iframe）で表示します。

## 商品モーダルでできること

| 操作 | 内容 |
|------|------|
| 商品情報の確認 | 画像・価格・説明・タグを確認できます（説明は「もっと見る」で全文表示） |
| バリエーション選択 | 色・サイズなどのバリエーションを選べます |
| 数量選択 | 購入数を選べます |
| カートに追加 | 選択した内容でECサイトのカートに追加します |
| 商品詳細へ | 商品ページ（URL）を開きます |

## 在庫について

在庫のないバリエーションは選べず、すべてのバリエーションを選ぶまでカート追加ボタンは押せません。

## 商品が表示できないとき

商品が下書きや削除済みの場合は、代わりの表示（サムネイル・タイトル・リンク）に切り替わります。
