# プレイリスト
{: .no_toc }

Tig Short の[コレクション](collection.html)やコンテンツを束ねて配信する「プレイリスト」を管理する画面です。ご利用にはプレイリスト機能の利用設定が必要です。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 画面の構成

- **一覧**: 作成済みのプレイリストを一覧表示します。
- **作成ウィザード**: 基本情報 → カテゴリ → コンテンツ／コレクション選択 → 確認、の順で作成します。

## できること

| 操作 | 内容 |
|------|------|
| 一覧 | プレイリストを一覧表示 |
| 新規作成 | ウィザードでプレイリストを作成 |
| 編集・保存 | 基本情報・カテゴリ・コンテンツ／コレクションを保存（段階保存・一括保存） |
| 複製 | 既存のプレイリストを複製 |
| 削除 | プレイリストを削除（複数まとめて削除も可能） |

## コレクションとの関係

プレイリストには、[コレクション](collection.html)やコンテンツをまとめて登録できます。作成ウィザードのコンテンツ／コレクション選択で追加します。
