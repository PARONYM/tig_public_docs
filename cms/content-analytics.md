# コンテンツアナリティクス
{: .no_toc }

Tig Video・Tig Short のコンテンツ（動画）1件ごとの視聴状況を分析する画面です。各コンテンツの詳細画面の「アナリティクス」から開きます（ライブ配信のアナリティクスとは別画面です）。

## 画面の構成

- **ヘッダー部分（画面上部）**: 期間の指定（全期間／期間指定・プリセット）、CSVダウンロード、（Short で承認機能が有効な場合）承認ボタン。
- **メイン領域（画面下部）**: 切り替え式の折れ線グラフと、各指標のセクション（再生数・再生デバイス・計測タグ別・再生完了率・タップ・ストック・URL遷移・アイテム別）。

以降は、各指標を説明します。

## 使い方

1. コンテンツ詳細を開き「アナリティクス」を選ぶ
2. 期間を指定して「表示」する（初期は直近1ヶ月）
3. 必要に応じて CSV をダウンロード、ヒートマップを開く

## 表示される指標

| 指標 | 内容 |
|------|------|
| 再生数 | 総再生数・5秒以上再生数・視聴UU（重複を除いた人数）。読み込み回数は管理者向け表示 |
| 再生デバイス | 視聴された OS の内訳（iOS／Android／Windows／Mac／その他） |
| 計測タグ別再生数 | 視聴URLに付けた[計測タグ（ta_tag）]({{ '/glossary.html' | relative_url }})ごとの再生数 |
| 再生完了率・平均再生率 | 動画の 25%／50%／75％／100% 地点まで再生された割合と、平均してどこまで再生されたか |
| タップ | 映像へのタップ総数（Tig領域／非Tig領域の内訳） |
| ストック・URL遷移 | アイテムのストック数・ジャンプ（リンク遷移）数（総数とユニーク） |
| アイテム別 | アイテムごとのストック・ジャンプ数（マルチリンクは子リンクごと） |

> 割合(%)の分母など細かな数え方は[用語集]({{ '/glossary.html' | relative_url }})も参照してください。

## ヒートマップ

動画上のどこがよくタップされたか（タッチ座標）を、動画に重ねて可視化できます。コンテンツ詳細の「ヒートマップ」から開きます。

## CSVダウンロード

- **イベント明細CSV**: 再生・タップ・ストック・ジャンプ等のイベントを1件ずつ出力。
- **Tig範囲CSV**: 各アイテムが動画上に表示される時間範囲。
- **LINE CSV**: 対応プランの場合に、LINEユーザー別のイベント履歴。
