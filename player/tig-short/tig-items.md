# 商品ポイント（Tigポイント）の表示とタップ
{: .no_toc }

Tig Short プレイヤーで、動画上に表示される商品ポイント（Tigポイント）と、そのタップで起きることの説明です。

## 目次
{: .no_toc .text-delta }

## Tigポイントとは

動画の中で商品が映る位置に合わせて表示される、**タップできる商品への入り口**です。動画の再生に合わせて位置が滑らかに移動し、吹き出しで商品名やサムネイルを表示できます。

## タップしたときの動き

タップしたときの動きは、CMS の**アイテム設定**で選べます。

| 設定 | タップ時の動き |
|------|----------------|
| 直接リンクへ | 商品ページなどのURLを別タブで開きます |
| ストック（あとで見る） | 「ストック」に保存します（ページ遷移はしません） |
| リンク＋ストック | ストックに保存しつつ、URLも開きます |
| ポップアップ（モーダル） | 商品情報をプレイヤー内のポップアップで表示します |

Shopify と連携している場合は、Shopify の商品情報をポップアップで表示できます。

## 表示のしかた（3つのタイプ）

商品ポイントは、コンテンツの設定によって次の表示になります。

| タイプ | 内容 |
|--------|------|
| 動画上のポイント（吹き出し） | 商品が映る位置に吹き出しで表示されます |
| カード型 | 紹介する商品が1件のとき、カード状に表示します |
| アイテム画像型 | 紹介する商品が2件以上のとき、画像を並べて表示します |

## ストックについて

タップやストック操作で保存した商品は「ストック（あとで見る）」に入り、端末に保存されます。ストックの一覧はストックボタンから確認できます。
