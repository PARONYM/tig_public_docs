## 公開設定の反映

管理画面の公開設定は、**閲覧ページを開いた時点で判定**されます。条件を満たさない場合、マガジンの中身は読み込まれません。

| 設定 | 効き方 |
|------|--------|
| 公開／非公開 | 非公開のマガジンは閲覧できません |
| 公開期間 | 期間外は閲覧できません |
| 非公開視聴パスワード | パスワードの入力を求め、一致した場合のみ閲覧できます |
| IPアドレス制限 | 許可した接続元からのみ閲覧できます |
| ドメイン制限 | 許可したサイトからのみ閲覧できます |

管理画面での編集内容が閲覧ページへ反映されるまでには、最大で10分程度かかる場合があります。

## 紙面の表示

ページ画像と、その上に置いたアイテムは、**元の紙面サイズに対する比率**で配置されます。画面サイズが変わっても位置関係が崩れません。

| 設定 | 内容 |
|------|------|
| 単ページ／見開き | 1ページずつ表示するか、2ページ並べて表示するか |
| 表示の固定 | 固定すると、閲覧者が表示方法を切り替えられなくなります |
| 表紙の扱い | 表紙を単独ページとして扱うかどうか（見開きの組み方が変わります） |
| ページ送り方向 | 右綴じ／左綴じ。スワイプやキー操作の向きもあわせて切り替わります |
| ページめくり演出 | スライド／めくり |

閲覧者が表示方法を切り替えた場合、その設定は次に開いたときも保持されます。

## 画面サイズの変化への追従

ブラウザの幅を変えたり端末を回転させたりすると、紙面とアイテムの位置が再計算されます。回転時は表示中のページを保ったまま読み込み直します。

## 書体

管理画面で指定した書体（Webフォント）を読み込み、アイテムの吹き出しに反映します。

## 埋め込みでの利用

マガジンは自社サイトへの埋め込みにも対応しています（全画面表示）。詳しくは[閲覧の記録と埋め込み]({{ '/player/tig-magazine/analytics-embed.html' | relative_url }})をご覧ください。
