# Tig Magazine コンテンツ一覧
{: .no_toc }

Tig Magazine（複数ページで構成する動画マガジン）のコンテンツを一覧で管理する画面です。検索・絞り込みのほか、新しいマガジンの作成、複製、削除ができます。

## 画面の構成

この画面は大きく2つの領域に分かれています。

- **ヘッダー部分（画面上部）**: 「Tig Magazine」の見出し、キーワード検索・全件表示、**新規アップロード**ボタン、件数表示、絞り込み（グループ〔代理店以上〕・ステータス・公開設定）、表示の切り替え（ボックス／リスト）、並べ替え。
- **メイン領域（画面下部）**: マガジンの一覧。各行にサムネイル・タイトル・アイテム数・状態（コピー中／変換中／変換エラー）・公開状態・登録日／更新日が表示され、行から詳細画面へ進めます。

以降は、各操作を説明します。

## 使い方

1. キーワードや絞り込み（グループ・ステータス・公開設定）で目的のマガジンを探す
2. 表示形式（ボックス／リスト）や並べ替えを切り替える
3. 行をクリックして詳細画面で編集する
4. 新しく作る場合は「新規アップロード」から作成する

## 新規作成（アップロード）

1. 「新規アップロード」を押す
2. ページ画像をまとめた ZIP ファイルをアップロードする
3. ページの順番を並べ替えて登録する
4. 変換処理が始まり、完了するとマガジンとして表示されます（変換中は一覧に「変換しています」と表示）

## 複製・削除

- **複製**: マガジンを設定・ページごと複製できます（複製中は「コピー中」と表示）。
- **削除**: マガジンを削除します（一覧から消えます）。

## ご利用の前提

- Tig Magazine は**マガジン機能が有効なグループ**でのみ利用できます。
