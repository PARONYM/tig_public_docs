## プレイヤーの読み込み

外部サイトへの埋め込み・視聴ページのどちらでも、必要なデータが読み込まれ次第プレイヤーが自動で起動します。読み込み中はローディング表示が出ます。

- 動画上のアイテム（Tigポイント）の位置情報は再生に合わせて順次読み込まれるため、長い動画でも先頭からすぐに視聴を始められます。
- 早送り・シークした場合も、移動先のアイテム情報が優先して読み込まれます。

## 起動時に指定できる主な設定

| 設定 | 内容 |
|------|------|
| 自動再生 | ページ表示と同時に再生を開始（ブラウザ仕様によりミュートで開始） |
| 開始位置 | 指定した秒数から再生を開始 |
| Tigポイントの初期表示 | アイテム表示の初期オン/オフ |
| ストック表示の方式 | ストック（お気に入り）UIの表示方法 |

## 対応環境

対応していないブラウザで開いた場合は、その旨のメッセージが表示されます。

## 公開制御・アクセス制限

コンテンツごとに、視聴できる相手を制限できます。

| 制限 | 内容 |
|------|------|
| 公開状態・公開期間 | 非公開・公開期間外のコンテンツは視聴できません |
| 視聴パスワード | パスワードの入力を求め、一致した場合のみ視聴できます |
| IPアドレス制限 | 許可した接続元からのみ視聴できます |
| ドメイン制限 | 許可したサイトに設置した場合のみ視聴できます |
| 非公開プレビュー | 公開前の確認用に、専用パスワードでの視聴を許可できます |

## プライバシー対応（GDPR）

EEA圏（欧州経済領域）からの視聴では、クッキー等の利用に関する同意画面が表示され、**同意までは計測やブラウザへの保存を行いません**。
