# クリックモーダル設置（HTML属性）
{: .no_toc }

HTML に属性を書くだけで、クリック時にプレイヤーをポップアップ表示できる設置方法です。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 特徴

- **JavaScript を書く必要がありません**。所定のクラスと属性を持つ要素を置き、スクリプトを読み込むだけで動作します。
- **製品を自動で判別**します。通常の動画・LIVE配信/アーカイブ・Magazine のいずれでも、同じ書き方で設置できます。

## 指定できること

| 指定 | 内容 |
|------|------|
| コンテンツの指定 | 表示するコンテンツを属性で指定 |
| プレイヤーの設定 | 属性に書いた設定がそのままプレイヤーに反映されます |
| 遅延読み込み | 開いたときに読み込む（ページ表示を軽くできます） |
| 自動オープン | ページを開いた時点でモーダルを自動表示（URLパラメータでの指定が優先） |
| パラメータ引き継ぎ | 設置ページのURLパラメータをプレイヤーへ引き継ぎ |
| カート導線 | カートのURLを指定するとカートアイコンを表示 |

## 見た目の調整

モーダルの幅・高さ・背景色・閉じるボタンの色や位置は、CSS の変数で調整できます。LIVE配信の場合はコメント欄の幅を含めてサイズが自動調整されます。
