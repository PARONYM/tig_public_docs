# Tig LIVE 配信一覧
{: .no_toc }

登録済みのライブ配信コンテンツを一覧・検索する画面です。各カードから「詳細」を開いて設定・配信操作を行います。

## 一覧の見方

配信コンテンツは**カード形式**で並びます。各カードでは次の内容が確認できます。

| 表示 | 内容 |
|------|------|
| サムネイル / タイトル | コンテンツの見出し |
| ステータス | 配信予約 / 配信中 / 配信終了 / アーカイブ などの状態 |
| 配信日時 | 配信の開始予定・実施日時 |
| 公開設定 | 公開 / 非公開 / 期間指定 のアイコン |
| ラベル | テスト・プレミア・アーカイブ取込 などの区分ラベル |

カードを選ぶと[配信詳細]({{ '/cms/tig-live/content-detail.html' | relative_url }})が開きます。

## 検索・絞り込み

次の条件で絞り込めます。設定した条件は、詳細画面へ移動して戻っても保持されます。

- **キーワード**: タイトル・説明などで検索
- **グループ**: 管理グループでの絞り込み（権限がある場合に表示）
- **ステータス**: 配信予約 / 配信中 / 配信終了 / アーカイブ
- **公開設定**: 公開 / 非公開 / 期間指定
- **アーカイブ取込**: 外部から取り込んだアーカイブの有無
- **並べ替え**: 配信日・登録日・更新日（昇順／降順）

## ステータスの種類

| ステータス | 意味 |
|-----------|------|
| 配信予約 | 配信開始前（予約済み） |
| 配信中 | ライブ配信中 |
| 配信終了 | ライブが終了した状態 |
| アーカイブ | 終了後に録画として視聴できる状態 |

## 複製

既存のライブ配信を**複製**して、新しい配信を素早く作成できます。配信前・配信済み・アーカイブのいずれからでも複製できます。

## ページ送り

一覧は一定件数ごとにページ送りで表示され、該当件数も確認できます。
