# アイテムのタップ挙動・モーダル・シェア
{: .no_toc }

LIVE配信の映像上に表示される紹介アイテムをタップしたときの挙動と、そこから開くモーダル・シェアについての概要です。

## Tigアイコン（吹き出し）の表示

紹介中のアイテムには、映像上に吹き出し（Tigアイコン）が表示されます。アイコンの種類・独自画像・大きさ・色・文字は管理画面から設定でき、アイテムごとに吹き出しを隠すこともできます。

LIVE配信では、映像の進行に合わせて**紹介中のアイテムが自動で切り替わり**、設定に応じて自動でモーダルを開くこともできます。

> LIVE配信では吹き出しの表示は常にオンで、視聴者による表示・非表示の切り替えはありません。

## 初回に表示される案内

- **フライヤー**: 視聴開始時に「Welcome to LIVE!」の案内が表示され、タップして配信に参加します（案内の裏では配信映像が流れています）。
- **ヘルプ**: 初回のみ操作の案内が表示され、その間は再生が一時停止します。

## タップしたときの挙動

アイテムの種類と挙動設定の組み合わせで決まります。

| 種類・設定 | タップしたときの挙動 |
|---|---|
| リンク（その場で遷移） | 別タブでリンク先を開きます（あわせてお気に入りに保存する設定も可能です） |
| リンク（モーダル） | ポップアップの中にリンク先を表示します |
| 商品（Shopify / Makeshop / ショップサーブ / エビスマート） | 商品モーダルを開きます |
| クーポン | クーポンモーダルを開きます |
| 記事 | 記事のモーダルを開きます |
| 投票 | 投票のモーダルを開きます |
| （上記以外） | お気に入りに保存します |

## モーダルの種類

タップしたアイテムに応じて、次のモーダルが開きます。

| モーダル | 内容 |
|---|---|
| 商品 | 商品画像・商品名・価格・在庫・バリエーション選択・数量・カート追加・商品説明 |
| クーポン | クーポンコードの表示とコピー |
| 記事 | 画像・タイトル・本文 |
| カルーセル | 複数画像をスワイプで切り替え |
| ページ表示 | 商品ページなどをモーダル内に表示 |
| 投票 | 選択肢への投票と結果表示（[投票・AIコーディネート]({{ '/player/tig-live/survey-ai.html' | relative_url }})） |

## シェア

配信のフッターにあるシェアボタンから、配信を共有できます。

- 対応端末では、端末標準の共有メニューが開きます。
- それ以外では、**リンクのコピー・X・LINE** から選べるメニューが表示されます。
