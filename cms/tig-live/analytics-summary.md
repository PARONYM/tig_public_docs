# Tig LIVE アナリティクス（LIVE配信）
{: .no_toc }

Tig LIVE の管理画面「アナリティクス」→ **LIVE配信タブ**で見られる各項目の意味をまとめます。
配信の到達（再生・視聴人数）、エンゲージメント（コメント・Like・ストック・ジャンプ）、視聴行動などを確認できます。

## 使い方

1. CMS で対象のライブコンテンツを開く
2. 「アナリティクス」→「**LIVE配信**」タブを選ぶ
3. 上部のグラフ・サマリー数値・各テーブルで状況を確認する

> 数値の多くは「視聴UU」を基準（分母）に割合(%)を併記します。用語は[用語集]({{ '/glossary.html' | relative_url }})も参照してください。

## 画面の構成（LIVE／アーカイブ／シーン分析）

アナリティクス画面の上部には、表示を切り替える要素があります。

- **LIVE／アーカイブ タブ**: 画面上部のタブで、**LIVE配信中の集計（本ページ）**と**アーカイブ（配信終了後）の集計**を切り替えます。アーカイブが無い配信では「アーカイブ」タブは選べず、LIVE のみ表示されます。
- **「シーン分析」ボタン**: 動画のシーン（時間帯）ごとの分析を別画面で開けます（→ 後述「シーン分析」）。

本ページは **LIVEタブ**の内容を説明します（アーカイブタブの各項目は別途）。

## 1. グラフ（時系列）

ライブ配信中の推移を1本の折れ線グラフで表示し、凡例で系列を切り替えられます。

| 系列 | 内容 |
|------|------|
| 視聴者数 | その時点の同時視聴者数の推移 |
| コメント / Like / タップ / ストック / ジャンプ | 各指標の時系列推移 |

- グラフ上部のボタンで**表示する指標を1つずつ切り替え**られ、それぞれの**合計値**も併記されます。
- 「視聴者数」は**同時視聴者数**の推移です。プレーヤー画面右上に出る累計視聴数とは別物です。
- グラフ下のボタンから **コメントログ**・**リアルタイムデータ** をダウンロードできます。

## 2. サマリー数値

| 項目 | 意味 |
|------|------|
| 総再生数 | 再生が開始された延べ回数 |
| 5秒以上再生数 | 5秒以上再生された延べ回数 |
| 視聴UU | 再生した人数（重複を除いたユニーク数） |
| 5秒以上視聴UU | 5秒以上再生した人数 |
| 新規視聴UU | 過去6ヶ月、同じグループのライブを一度も視聴していない新規視聴者数 |
| ライブ終了後UU | ライブ終了後の画面（事後フィラー）を訪れた人数 |
| ピーク視聴UU | 配信中の同時視聴者数の最大値 |
| 再生デバイス（UU） | 視聴者のOS内訳（iOS / Android / Windows / Mac / その他）の円グラフ |
| load数 | プレーヤーの読み込み回数（管理者向け表示） |

## 3. コメント・Like・ストック・ジャンプ（総数）

「総数」と「視聴UUに対する割合(%)」の2行で表示します。

| 項目 | 意味 |
|------|------|
| コメント | コメント投稿の総数 |
| Like | 「いいね」の総数 |
| ストック | アイテムをストックした延べ人数（アイテム別の合計） |
| ジャンプ | リンク遷移（ジャンプ）の総数 |

## 4. タップ分析

| 項目 | 意味 |
|------|------|
| 総タップ回数 | 映像領域へのタップ総数（再生1回あたりの平均も表示） |
| Tig領域タップ | アイテム等のTig UIへのタップ |
| 非Tig領域タップ | それ以外の映像領域へのタップ |
| ストック・ジャンプ回数分布 | 「1人あたり何回ストック／ジャンプしたか」の人数分布グラフ |

## 5. 配信時間の分析（LIVE配信のみ）

| 項目 | 意味 |
|------|------|
| 配信時間 | 実際に配信された時間 |
| 平均視聴継続時間 | 1人あたりの平均視聴時間 |
| ～5秒 / ～10秒 / ～30秒 / ～1分 | 短時間で離脱した人数の分布 |
| 視聴継続時間分布（グラフ） | 1分以上の視聴時間別の人数分布 |

> 視聴継続時間は「再生」とは別の計測（視聴中の接続時間）に基づくため、各区間の人数を合計しても視聴UUとは一致しません。

## 6. 計測タグ別（ta_tag）

視聴URLに付けた**計測タグ（ta_tag）**ごとの結果です。

| 項目 | 意味 |
|------|------|
| 視聴UU / ストック / ジャンプ | 計測タグ別の数値 |
| 計測タグ別 再生数（円グラフ） | タグ別の視聴UUを可視化（11個目以降は「その他」、タグなしは「計測タグなし」） |

> 同じ人が別タグのURLから視聴すると各タグに数えられるため、タグ別の合計が全体の視聴UUを超えることがあります。

## 7. アイテム別

| 項目 | 意味 |
|------|------|
| 紹介アイテム数 | ライブ中に紹介したアイテム数 |
| アイテム別 ストック / ジャンプ | アイテムごとのストック数・ジャンプ数 |
| マルチリンク ジャンプ | マルチリンクの子リンクごとのジャンプ数 |

> ストック数・ジャンプ数の多い順に並び、初期表示は上位5件です（「全件表示」で展開）。

## 8. ユーザー行動別

「UU（人数）」と「視聴UUに対する割合(%)」の2行で表示します。

| 項目 | 意味 |
|------|------|
| コメント（UU） | コメントを投稿した人数 |
| ストックアクション（UU） | 1回以上ストックをした人数 |
| ジャンプアクション（UU） | 1回以上ジャンプをした人数 |
| 視聴行動UU合計 | ストック・ジャンプのいずれかを1回以上行った人数 |

## 9. ライブ開始前の行動（LIVE配信のみ）

| 項目 | 意味 |
|------|------|
| 事前告知UU | 開始前の事前フィラー（待機画面）を訪れた人数 |
| お知らせUU | 開始前に、お知らせ受取（Googleカレンダー／LINE／メール）のいずれかをタップした人数 |
| 過去アーカイブ誘導UU | 開始前に「過去のライブ」からアーカイブをタップした人数 |

## 10. ライブ中の特殊行動

| 項目 | 意味 |
|------|------|
| クーポン取得UU | クーポンコードをコピーした人数 |
| 配信者メニュー表示 | 「配信者メニュー」をタップした人数 |

## シーン分析

画面上部の「**シーン分析**」ボタンから開く、別画面の分析です。動画を見ながら、**シーン（再生位置・時間帯）ごとの推移**を確認できます。

- **期間**: 全期間／期間指定（カレンダー）／過去1日〜1ヶ月から選べます。
- **LIVE／アーカイブ 切替**: シーン分析の中でも切り替えられます。
- **表示できる推移（凡例で切替）**:
  - LIVE: 視聴者 / コメント / タップ / ストック / ジャンプ
  - アーカイブ: 再生完了率 / タップ / ストック / ジャンプ

## CSVダウンロード

画面から以下の4種類をダウンロードできます。

| CSV | 内容 |
|-----|------|
| イベントログ | 再生・タップ・ストック・ジャンプ等のイベント1件ずつ |
| リアルタイムデータ | 毎分の同時視聴・Like 等のスナップショット |
| コメント | コメント1件ずつ（NGコメントは除外済み） |
| 滞在時間 | 視聴者ごとの接続時間 |

## 補足（数え方）

- **UU** は端末（ブラウザ）単位で数えます。同じ方でも端末・環境が変わると別の視聴者として数えられる場合があります。
- 割合(%)は原則「**視聴UU**」を分母にします。
- コメント・Like・ピーク視聴UU などのリアルタイム集計が未集計のときは「-」と表示されます。
