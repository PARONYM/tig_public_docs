# 閲覧の記録と埋め込み
{: .no_toc }

## 記録される行動

| 記録項目 | 内容 |
|---------|------|
| ページ閲覧 | どのページが読まれたか。見開き表示のときは並んでいる2ページ分を記録します |
| 閲覧開始 | 読み始め |
| アイテムのタップ | どのアイテムがタップされたか（紙面の位置も記録します） |
| アイテムへの注目 | アイテムに触れた／近づけた操作 |
| お買物メモへの保存 | どの商品が保存されたか |
| リンク先への遷移 | どのリンクから外部サイトへ移動したか |
| 閲覧終了 | 読み終わり |

集計結果は管理画面の[アナリティクス]({{ '/cms/tig-magazine/analytics.html' | relative_url }})で確認できます。タップ位置はヒートマップとしても集計されます。

## 同じページを何度も見た場合

行き来して同じページに戻っても、**ページ閲覧は重複して記録されません**。端末を回転させて画面が読み込み直された場合も、二重に記録されないよう調整しています。

## 記録しない・記録が始まらない場合

| 状況 | 扱い |
|------|------|
| 表紙（ポスター）を表示している間 | 本編に入るまでページ閲覧の記録は始まりません |
| 初回ガイダンスを表示する前 | ページ閲覧の記録は始まりません |
| 内部プレビュー（`ipv` 付きのURL） | 内部からの閲覧として区別して記録します |

## 外部の計測ツールとの連携

閲覧開始のタイミングは Google アナリティクス（GA4）にも送信されます。それ以外の行動は Tig の集計基盤に記録されます。

## 自社サイトへの埋め込み

マガジンは自社サイトに埋め込んで表示できます。

- **対応している表示形式は「全画面」のみ**です。ページ全体をマガジンにする形になります（フローティング表示やスクロール追従表示には対応していません）。
- 設置したページのURLパラメータは、そのままマガジン側へ引き継がれます。
- コンバージョン計測タグとの連携に対応しています。

埋め込み全体の考え方は[埋め込み連携の全体像]({{ '/player-common/embed-overview.html' | relative_url }})をご覧ください。

### 埋め込んだページとの連携

埋め込み表示のとき、マガジンから設置ページへ次の情報を通知できます。

| 通知 | 内容 |
|------|------|
| 準備完了 | マガジンの表示準備が整ったこと |
| ページ変更 | 現在表示しているページ番号 |
| お買物メモへの保存 | 保存された商品の情報 |

設置ページ側から**表示するページを指定**することもできます。
