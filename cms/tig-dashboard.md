# Tig ダッシュボード
{: .no_toc }

コンテンツの種別（Tig Video／Magazine／Tig Branch Video／Tig LIVE／Tig Short）を切り替えて、**視聴の状況**（再生・タップ・ストック・URL遷移・コメント・Like・再生完了率など）をまとめて確認できる画面です。

## 画面の構成

- **絞り込み（画面左／上部）**: コンテンツの種別・期間・グループ・コンテンツの指定で表示対象を絞り込みます。
- **メイン領域**: 「全体」「コンテンツ個別」「商品アイテム個別」をタブで切り替えて表示します。
- **ダウンロード**: 表示中の絞り込み条件で CSV／ZIP をダウンロードできます。

## 絞り込み

| 項目 | 内容 |
|------|------|
| 種別 | Tig Video／Tig Magazine／Tig Branch Video／Tig LIVE／Tig Short のいずれかを選択 |
| 期間 | 開始日・終了日を指定（プリセットからの選択も可。初期は直近1か月） |
| グループ | 対象グループで絞り込み（権限のあるグループの範囲で表示） |
| コンテンツ | 「全て」「指定したコンテンツのみ」「指定したコンテンツを除外」から選択（コンテンツIDを指定） |
| LIVE／アーカイブ | 種別が Tig LIVE のとき、ライブ視聴分／アーカイブ視聴分を切り替え |

## タブ（表示できる内容）

| タブ | 内容 |
|------|------|
| 全体 | 期間内の指標サマリと、日次の推移グラフ |
| コンテンツ個別 | コンテンツごとの再生・タップ・ストック・URL遷移・再生完了率などの一覧（各列で並び替え） |
| 商品アイテム個別 | アイテムごとの視聴・ストック・URL遷移などの一覧（商品名・商品コードを含む） |

## ダウンロード

| メニュー | 内容 |
|---------|------|
| コンテンツ一覧 | コンテンツ別の集計を CSV でダウンロード |
| コンテンツ個別 | コンテンツごとのデータを ZIP でまとめてダウンロード |
| 商品アイテム一覧 | アイテム別の集計を CSV でダウンロード |
| LINEアイテムリスト | LINE 連携をご利用の場合に、アイテムリストを CSV でダウンロード |

CSV は表示中の絞り込み条件と同じ範囲で出力されます。
