# パスワード変更
{: .no_toc }

ログイン中のご自身のパスワードを変更する画面です。画面右上のユーザーメニューの「パスワード変更」から開きます。

## 変更の手順

1. 「現在のパスワード」を入力します。
2. 「新しいパスワード」を入力します。入力中に、条件を満たしているかが画面上に表示されます（パスワードの強度も確認できます）。
3. 「新しいパスワード（確認）」に同じものを入力します。
4. 「パスワード変更」をクリックします。「パスワードを変更しました。」と表示されたら完了です。

## 新しいパスワードの条件

- 8文字以上、32文字以下
- 半角英数字または記号【~!@#$%^()_-+{}[]:',.?/】
- 1文字以上の小文字、大文字、数字を含める
- 【推奨】1文字以上の記号を含める

※メールアドレスと同じ文字列はパスワードに設定できません。

## パスワード変更のお願いが表示される場合

現在のパスワードポリシーが定められる前に設定したパスワードを使い続けている場合、ログイン後に「パスワードポリシーの変更に伴い、パスワードの変更をお願いします」という画面が表示されます。

- 新しい条件を満たすパスワードに変更すると、通常どおり利用を再開できます。
- 変更せずに閉じた場合、次回アクセス時にも同じ画面が表示されます。

## パスワードを忘れた場合

現在のパスワードが分からない場合は、ログイン画面の「パスワードを忘れた方」から再設定してください。詳しくは[パスワード再発行・ロック解除]({{ '/cms/reissue.html' | relative_url }})を参照してください。
