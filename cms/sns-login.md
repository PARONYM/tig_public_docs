# SNSログイン連携設定
{: .no_toc }

SNS（Facebook／Instagram／TikTok）アカウントとの連携を設定する画面です。ヘッダーの「SNSログイン連携設定」から開きます。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## できること

| 連携 | 内容 |
|------|------|
| Facebookログイン連携 | CMSのアカウントに Facebook を紐付け、Facebook でログインできるようにします |
| Instagram連携 | Instagram アカウントを連携します（動画の取り込みに使用） |
| TikTok連携 | TikTok アカウントを連携します（動画の取り込みに使用） |

## 連携の使いみち

Instagram／TikTok を連携すると、**Tig LIVE のアーカイブ**や **Tig Short** で、そのSNSの動画を取り込めるようになります（取り込み自体は各作成画面から行います）。
