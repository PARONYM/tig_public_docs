# EC ダッシュボード
{: .no_toc }

Tig 経由の視聴が、どれだけ **EC（購買）の成果**につながったかを横断的に分析する画面です（LIVE／Tig Video／Tig Short をまたいで集計）。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 画面の構成

- **ヘッダー部分（画面上部）**: 期間の指定（対比期間も指定可）、グループ・タグでの絞り込み、CSVダウンロード。
- **メイン領域（画面下部）**: 「サマリ」「コンテンツ別」「アイテム別」をタブで切り替えて表示します。

> 利用には EC 成果の計測連携の準備が必要です（準備が完了していない場合は表示されません）。

## 表示される内容

| タブ | 内容 |
|------|------|
| サマリ | 期間全体の PV・視聴数・**直接購入／間接購入**（売上金額・購入品数・購入率）。対比期間との比較も表示 |
| コンテンツ別 | コンテンツごとの視聴数・購入金額・購入品数・平均視聴時間・ストック・URL遷移 |
| アイテム別 | 購買の明細（購入日時・直接／間接・商品名・EC商品コード・金額・品数・視聴したコンテンツ・視聴経路） |

## 直接購入と間接購入

- **直接購入**: 視聴後すぐに購入されたもの。
- **間接購入**: 視聴・クリックの後、後日購入されたもの（表示するかは設定で切り替えられます）。

## フィルタ・CSV

- **期間／対比期間・グループ・タグ**で絞り込めます。
- サマリ・コンテンツ別・アイテム別はそれぞれ **CSV ダウンロード**できます。
