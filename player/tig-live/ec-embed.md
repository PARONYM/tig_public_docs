# 外部サイトへの埋め込み・フローティング
{: .no_toc }

LIVE配信・アーカイブを、EC サイトなどの外部サイトに埋め込んで表示する機能の概要です。

## 埋め込みの種類

| 種類 | 内容 |
|------|------|
| フローティング | ページ上に小窓を固定表示。プレビュー動画が再生され、タップで本編がポップアップで開く |
| ページ内埋め込み | ページ内の指定位置に表示し、スクロールで画面外に出ると隅に追従表示（ピクチャーインピクチャー） |
| アーカイブ一覧 | 過去のアーカイブをスライダー・タイル形式で一覧表示 |
| 全画面 | ページ全体でプレイヤーを表示 |

## 配信の状態に合わせた表示

- **配信前**: 次回配信の告知（サムネイル＋LIVEアイコン）を表示できます。
- **配信中**: ライブ映像そのものを小窓で流すこともできます（設定時）。
- **配信終了後**: アーカイブの案内に自動で切り替わります。

## 商品ページとの連携

- 商品コードを指定すると、**その商品を紹介しているアーカイブ**が自動で選ばれて表示されます（複数コードの指定も可能）。
- 本編視聴中のカート連携や、気になる商品を保存する「お買物メモ」も利用できます。
- ページ遷移でURLが変わるサイト（SPA）にも対応しています。

## 計測

埋め込みの表示・再生・クローズは計測され、埋め込み先ページの計測ツール（GA等）にもイベントを連携できます。
