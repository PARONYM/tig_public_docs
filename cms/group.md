# グループ管理
{: .no_toc }

コンテンツやユーザーを束ねる「グループ」を管理する画面です。ヘッダーの「その他管理」から開きます。グループは親子の階層を持てます。

{: .no_toc .text-delta }

## できること

| 操作 | 内容 |
|------|------|
| 一覧・検索 | グループを一覧表示・検索 |
| 登録 | 親グループ・ロール（権限）などを指定して新規追加 |
| 編集 | グループ情報の変更（親グループ・IPアクセス制限など） |
| 削除 | グループを削除（過去データは保持されます） |
| サーバートークン発行 | LINE 連携をご利用の場合に必要なトークンを発行 |
| 利用状況 | グループの製品別の利用状況（コンテンツ数）を確認 |

## IPアクセス制限

グループ単位で、アクセスを許可／拒否する IP アドレス（範囲）を設定できます。

## 利用状況

各グループについて、Tig Video／Tig Magazine／Tig Branch Video／Tig LIVE（アーカイブ含む）／Tig Short の利用状況（コンテンツ数など）をまとめて確認できます。

## ロール（権限）

グループには[ロール（権限）](roles.html)を設定します。グループのロールが、所属ユーザーに割り当てられる権限の範囲を決めます。
