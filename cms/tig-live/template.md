# Tig LIVE 配信テンプレート
{: .no_toc }

ライブ配信を新規作成するときに、各種設定をまとめて引き継ぐための[配信テンプレート]({{ '/glossary.html' | relative_url }})を管理する画面です。サイドバーの「全体設定」→「テンプレート管理」から開きます。

{:toc}

## できること

| 画面 | 内容 |
|------|------|
| 一覧 | 登録済みテンプレートの一覧・並べ替え・削除。「テンプレートを作成」から新規へ |
| 詳細・編集 | テンプレート／ライブ／画像／アイテム／コンテンツ連携／オプション のタブで内容を編集 |
| 新規作成 | ステップに沿って作成。既存のライブ配信を選んで、その設定を雛形として読み込むこともできます |

## テンプレートに含められる設定

タイトル・説明、配信時間、アイテムの並べ方、フィラーのテキスト・画像、ロゴ、配信者、コンテンツ連携（カレンダー・LINE・メールマガジン等）、字幕、タグ、URLパラメータなど。

作成したテンプレートは、[配信の新規作成]({{ '/cms/tig-live/content-create.html' | relative_url }})の「通常作成」でテンプレートを選ぶと、これらの設定が引き継がれます。
