# 用語集

本サイトで使う用語をまとめています。各機能ページの「用語」欄は、この用語集からの抜粋です。
用語の追加・更新は、原則このページ（用語集）を正として行います。

| 用語 | 意味 |
|------|------|
| UU（ユニークユーザー） | 重複を除いた利用者の数 |
| 視聴 | 動画の再生が開始されたこと |
| 総再生数 | 再生が開始された延べ回数（同じ人による複数回もそれぞれ数える） |
| 5秒以上再生数 | 5秒以上再生された延べ回数 |
| 視聴UU | 再生した人数（重複を除いたユニーク数） |
| 新規視聴UU | 過去6ヶ月、同じグループのライブを一度も視聴していない新規の視聴者数 |
| Like | 視聴者が送る「いいね」 |
| コメント | ライブ配信中に視聴者が投稿するコメント |
| リアルタイム推移 | ライブ配信中の各指標（視聴者・コメント・Like など）の時系列の変化 |
| ライブ配信 | リアルタイムで配信される動画。配信中の指標を計測できる |
| アーカイブ配信 | ライブ配信の終了後に、録画として視聴できる動画 |
| グループ | コンテンツをまとめる単位。新規視聴UU の判定範囲などに使われる |
| ピーク視聴UU | 配信中の同時視聴者数の最大値 |
| ライブ終了後UU | ライブ終了後の画面（事後フィラー）を訪れた人数 |
| タップ | 映像領域をタップ（タッチ／クリック）する操作 |
| Tig領域 | 映像上で、アイテム等の Tig の UI が表示される領域 |
| ストック | アイテムを後で見るために保存すること |
| ジャンプ | アイテムのリンク先URLへ遷移すること |
| 計測タグ（ta_tag） | 流入元を区別するためにURLへ付与する計測用のタグ |
| 事前フィラー | ライブ開始前に表示される待機画面 |
| 事後フィラー | ライブ終了後に表示される画面 |
| 配信者メニュー | ライブ中に表示される、配信者の情報・リンクのメニュー |
| マルチリンク | 1つのアイテムに複数のリンクを設定する機能 |
| 視聴行動UU合計 | ストック・ジャンプのいずれかを1回以上行った人数 |
| 紹介アイテム数 | ライブ中に紹介したアイテムの数 |
| 再生UU | 再生した人数（重複を除いたユニーク数。アーカイブ配信では「視聴UU」をこの表記にしている） |
| 再生完了率 | 動画のどこまで再生されたか。25% / 50% / 75% / 100% の各地点まで再生した割合 |
| 平均再生率 | 平均して動画のどれくらいの割合まで再生されたか |
