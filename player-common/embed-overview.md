# 埋め込み連携の全体像
{: .no_toc }

自社サイトへの Tig プレイヤーの設置と、自社EC との連携についての全体像です。

1. TOC

## できること

自社サイトのページにスクリプトを設置すると、その場に Tig のプレイヤーが表示されます。商品ページであれば、**その商品を紹介している動画を自動で選んで表示**したり、カートへの導線をつないだりできます。

## 表示形式

| 形式 | 内容 |
|------|------|
| フローティング | ページ上に小窓を固定表示。プレビュー動画が流れ、タップで本編が開く |
| ページ内埋め込み | 指定した位置に表示。スクロールで画面外に出ると隅に追従（PiP） |
| クリックモーダル | サムネイルやボタンのクリックでポップアップ表示 |
| 一覧表示 | 複数のコンテンツをスライダー／タイル形式で一覧 |
| 全画面 | ページ全体にプレイヤーを表示 |

対応する形式は製品によって異なります。詳しくは各製品のプレイヤーのページをご覧ください。

## 商品との連携

- 商品ページに**商品コード**を渡すと、その商品を紹介するコンテンツが自動で選ばれます（複数コードの指定も可能）。
- 本編視聴中のカートへの導線、気になる商品を保存する「お買物メモ」も利用できます。

## ECサービスとの連携状況

| ECサービス | 商品情報の取り込み | プレイヤー側の連携 |
|-----------|------------------|------------------|
| Shopify | ○ | ○（商品詳細の表示・カート追加） |
| Makeshop | ○ | 表示のみ |
| ショップサーブ | ○ | 表示のみ |
| エビスマート | ○ | ○（カート投入の連動表示） |

上記以外のECサービスをご利用の場合も、汎用の連携方法（視聴イベントの受け取り）でカート連携を実装できます。

## 計測

- 埋め込み経由の表示・再生・クローズは Tig 側で計測されます。
- お客様サイトの解析ツール（GA等）へイベントを連携できます。
- コンバージョン計測タグとの連携にも対応しています。

## 設置方法の種類

設置方法は、スクリプトを呼び出す方式・HTMLの属性で指定する方式・専用タグを使う方式の3通りがあります。ご要望に応じて弊社で埋め込みコードを作成しお渡ししています。
