# Tig LIVE 配信の新規作成
{: .no_toc }

新しいライブ配信コンテンツを作成する画面です。配信一覧の「新規作成（コンテンツを作成）」から、3つの作成方法を選べます。

## 3つの作成方法

| 作成方法 | 用途 |
|---------|------|
| **通常作成** | これからライブ配信する新しいコンテンツを作る |
| **アーカイブから作成** | 既存の録画動画（アップロード／Instagram取り込み）を取り込んでアーカイブ配信を作る |
| **プレミアから作成** | 事前に用意した動画を、指定した時刻に自動でライブ開始する（[プレミア公開]({{ '/glossary.html' | relative_url }})） |

> 「アーカイブから作成」「プレミアから作成」は、利用しているグループの設定で有効な場合に表示されます。

## 通常作成

次の項目を設定して「ライブコンテンツを作成」します。

| 項目 | 必須 | 内容 |
|------|------|------|
| テンプレート | 任意 | 配信テンプレートを選ぶと、各種設定を引き継いで作成できます |
| タイトル | 必須 | コンテンツの見出し |
| 配信開始日時 | 必須 | 配信を始める日時（少し先の日時を指定します） |
| 共通設定画像 | 任意 | サムネイル・SNS共有画像・フィラーに共通で使う画像 |
| ロゴ画像 | 任意 | プレーヤーに表示するロゴ |

作成すると配信詳細画面が開き、続けて細かい設定ができます。

## アーカイブから作成

すでにある録画動画を取り込んで、アーカイブ配信として作成します。

- **動画**: アップロード、または Instagram から取り込み
- **サムネイル**: 任意（アップロード時は自動生成されます）
- **タイトル**: 必須
- **配信した日時**: 過去の日時を指定します
- **アイテムの並べ方**: 作成後は変更できないため、ここで決めます

> 動画は一定の長さ以上が必要です。

## プレミアから作成

事前に用意した動画を、指定時刻に自動でライブ開始する作成方法です。

- **動画**: 必須（MP4／MOV）
- **サムネイル**: 必須
- **タイトル**: 必須
- **配信開始日時**: 必須（現在時刻より一定時間あとを指定します）
- **アイテムの並べ方**: 作成後は変更できません

作成後は予約状態になり、指定時刻になると自動でライブが始まります。配信終了後は自動でアーカイブ化することもできます。
