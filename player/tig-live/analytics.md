# 視聴の記録（計測）

## 記録される行動

| 記録項目 | 内容 |
|---------|------|
| 再生 | 再生開始・一定時間の視聴・視聴終了 |
| アイテム操作 | タップ・お気に入り保存・リンク先への遷移 |
| 商品・クーポン | 商品モーダルの表示・カート追加・クーポンの表示とコピー |
| シェア | 配信のシェア |
| メニュー | 紹介アイテム履歴・出演者情報メニューの表示 |
| 視聴区間 | どの区間を視聴したか（離脱・復帰も含む） |

> コメント・いいね・投票・視聴者数は、これらとは別のしくみでリアルタイムに記録されます。

## 記録しない場合

非公開プレビュー（パスワード付きの確認用表示）では、視聴の記録は行われません。

## 外部の計測ツールとの連携

再生開始のタイミングは Google アナリティクス（GA4）にも送信されます。それ以外の行動は Tig の集計基盤に記録されます。

## 視聴区間の記録

どの区間を視聴したかを記録します。別のタブに切り替えると記録をいったん確定し、戻ると再開するため、離脱をまたいでも記録が欠けません。
