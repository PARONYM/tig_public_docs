# その他管理（連携）
{: .no_toc }

「その他管理」メニューの「インテグレーション」画面です。外部サービスとの連携やプレーヤーの配信に関わるグループ設定を行います。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## できること

| 項目 | 内容 |
|------|------|
| 計測連携の状態 | 購買計測（EC計測）の連携状態を確認し、「確認依頼」を送れます |
| 埋め込み許可ドメイン | プレーヤーを埋め込めるドメインを登録・変更・削除します |
| クライアントURL | 独自ドメインで配信する場合のURL（LIVE／Short）を設定します |

## 埋め込み許可ドメイン

プレーヤーを設置するサイトのドメインを登録します。登録したドメインでのみ埋め込みが有効になります（登録内容に応じて必要な認可情報が発行されます）。

## クライアントURL

独自ドメインでの配信をご利用の場合に、配信用のURLを設定できます（LIVE・Tig Short それぞれ）。
