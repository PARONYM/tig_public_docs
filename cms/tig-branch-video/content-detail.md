# Tig Branch Video コンテンツ詳細
{: .no_toc }

Tig Branch Video（分岐動画）の設定・編集を行う画面です。タブを切り替えて、動画ノードの追加・分岐のつなぎ込み・各種設定を行います。

## 画面の構成

この画面は大きく2つの領域に分かれています。

- **ヘッダー部分（画面上部）**: サムネイル、プレビュー、公開URL、複製・削除、アナリティクスへの導線、状態表示（変換中・変換エラー）。
- **メイン領域（画面下部）**: タブ（基本設定・チャプター設定・動画設定・詳細設定・Tigアイコン設定・公開設定）を切り替えて編集します。**動画設定**タブが分岐動画の中心で、動画ノードの一覧・追加・導入動画の指定を行います。**分岐構成図**で全体のつながりを確認できます。

以降は、各タブでできることを説明します。

## メイン領域での操作（タブ）

| タブ | 内容 |
|------|------|
| 基本設定 | タイトル・説明・SEO・所有グループ・備考・カスタムデータ |
| チャプター設定 | チャプターの表示 |
| 動画設定 | **分岐動画の中心**。動画ノードの一覧・追加、導入動画（最初に再生する動画）の指定、ホームボタン・遷移演出 |
| 詳細設定 | 自動再生・ミュート・ループ・開始位置・コントロールバー |
| Tigアイコン設定 | アイコンの種類・画像・サイズ・色・不透明度・省略表示 |
| 公開設定 | 公開モード（公開／非公開／期間指定）・視聴パスワード・IP／ドメイン制限 |

## 動画の追加（3つの方法）

動画設定タブから、分岐動画を構成する動画ノードを追加できます。

1. **既存の Tig Video から追加**: ライブラリの動画を選んで分岐動画として取り込みます。
2. **新規アップロード**: 動画をアップロードして追加します。
3. **削除済みから復元**: 一度削除した動画を復元して追加します。

最初に追加した動画は**導入動画**（最初に再生される動画）になります。導入動画は動画設定タブで切り替えられます。

## 分岐のつなぎ込みと分岐構成図

動画ノードどうしを分岐でつなぎ、視聴者の選択で再生が枝分かれするインタラクティブ動画を作れます。**分岐構成図**で、どの動画からどの動画へつながっているか（導入動画・各ノード・分岐）を一覧で確認できます。各ノードをクリックすると、その動画の詳細画面へ進めます。
