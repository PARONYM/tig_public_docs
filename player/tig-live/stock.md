# ストック（お気に入り）
{: .no_toc }

LIVE配信で気になったアイテムを保存し、あとから見返せる「ストック（お気に入り）」についての概要です。

## 追加のしかた

紹介アイテムをタップするとストックに追加されます。追加のときは、スマートフォンではアイテムがストックフォルダへ吸い込まれるアニメーション、パソコンではストック一覧が自動で開いてスライドで追加される演出が表示されます。

## 一覧・詳細・削除

- ストックは**新しい順**に一覧表示されます（スマートフォンは画面下から、パソコンは画面右端から開きます）。
- 一覧のアイテムをタップすると、リンク先や商品モーダルなどの詳細を開けます。
- アイテムを削除でき、削除するとお知らせ（トースト）が表示されます。すべて削除すると一覧は自動的に閉じます。
- 1つのアイテムに複数のリンクがある場合は、一覧から各リンクを開けます。

## 保存のされ方

ストックの保存先は、視聴のしかたによって変わります。

| 視聴のしかた | 保存先 |
|------|------|
| Tig の視聴ページを直接開いた場合 | ご覧のブラウザ内に保存され、再訪時に復元されます |
| 自社サイトへ埋め込んで表示した場合 | 自社サイト側でコンテンツをまたいで共有できる形で保存されます |
| LINE から視聴した場合 | LINE アカウントにひもづけて保存され、別の端末でも復元されます |

いずれの場合も、保存されるのはアイテムの情報のみで、個人情報は含まれません。
