# 用語集

本サイトで使う用語をまとめています。各機能ページの「用語」欄は、この用語集からの抜粋です。
用語の追加・更新は、原則このページ（用語集）を正として行います。

| 用語 | 意味 |
|------|------|
| UU（ユニークユーザー） | 重複を除いた利用者の数 |
| 視聴 | 動画の再生が開始されたこと |
| 総再生数 | 再生が開始された延べ回数（同じ人による複数回もそれぞれ数える） |
| 5秒以上再生数 | 5秒以上再生された延べ回数 |
| 視聴UU | 再生した人数（重複を除いたユニーク数） |
| 新規視聴UU | 過去6ヶ月、同じグループのライブを一度も視聴していない新規の視聴者数 |
| Like | 視聴者が送る「いいね」 |
| コメント | ライブ配信中に視聴者が投稿するコメント |
| リアルタイム推移 | ライブ配信中の各指標（視聴者・コメント・Like など）の時系列の変化 |
| ライブ配信 | リアルタイムで配信される動画。配信中の指標を計測できる |
| アーカイブ配信 | ライブ配信の終了後に、録画として視聴できる動画 |
| グループ | コンテンツをまとめる単位。新規視聴UU の判定範囲などに使われる |
