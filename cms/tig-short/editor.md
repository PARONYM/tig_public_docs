# Tig Short 動画編集（β）
{: .no_toc }

Tig Short の動画を、CMS の画面上（ブラウザ）で編集できる機能です。トリミング・動画の追加・音声の後付け・字幕の焼き込みができます。**現在ベータ版（β）**として提供しています。

## 画面の構成

動画編集は、専用の**全画面エディター**で行います。起動するボタンは2か所にあります。

- **コンテンツ一覧の「動画を編集して作成」**: ゼロから（素材をアップロードして）新しい Short 動画を作成します。従来の「新規作成」（撮影済み動画のアップロード）はそのまま残っており、用途で使い分けられます。
- **コンテンツ詳細の「動画編集(β)」**: すでに登録済みの動画を読み込んで編集します（動画が未登録のコンテンツでは押せません）。
- **エディター画面（全画面）**: ボタンを押すと、編集用のエディターが画面いっぱいに開きます。編集が終わったら保存すると、動画が反映されます（編集をやめる場合は破棄の確認が出ます）。

以降は、できることと使い方を説明します。

## できること

| 機能 | 内容 |
|------|------|
| トリミング | 動画の不要な部分を切り取る |
| 動画の追加 | 複数の動画クリップをつなげて1本にする |
| 音声の後付け | BGM・音声ファイルを重ねる |
| 字幕 | 字幕を入れて動画に焼き込む |

## 使い方

### 既存の動画を編集する

1. コンテンツ詳細を開く
2. 「動画編集(β)」を押す（動画が登録済みの場合）
3. エディターで編集し、保存する
4. 編集後の動画に差し替わります（タイトルもエディターと同期します）

### ゼロから作る

1. コンテンツ一覧で「動画を編集して作成」を押す
2. エディターで素材をアップロードして編集し、保存する
3. 新しいコンテンツが作成され、詳細画面が開きます

## ご利用上の注意（β）

- 本機能は**ベータ版**です。今後、仕様・UI が変わる場合があります。
- 動画の長さには上限・下限があります（短すぎ・長すぎる動画は保存できません）。
- 同じコンテンツを複数人で同時に編集した場合、後から保存した内容で上書きされることがあります。
