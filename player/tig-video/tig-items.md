## アイテムの表示

動画内の商品や人物に合わせて、アイテム（Tigポイント）が表示されます。アイテムは映像の動きに**滑らかに追従**し、タップすると詳細の確認やストックができます。

## マーカーの種類

| 種類 | 内容 |
|------|------|
| サークル | 対象物の位置を点滅するサークルで示す |
| 吹き出し | アイテム名などの文言付きで表示。色・大きさ・向き・フォントを設定できます |
| サムネイル | アイテム画像をマーカーとして表示 |
| 独自画像 | 任意の画像をマーカーとして表示 |
| カウントダウン・ゲーム | カウントダウン表示や、並び替えゲームなどの企画向け表示 |

## 表示のオン/オフ（Tigモード）

- 切り替えボタンで、アイテム表示を一括でオン/オフできます（Tigモード）。
- 初期状態やボタンの表示有無は、埋め込み時の設定で制御できます。切り替えた状態は次回の視聴にも引き継がれます。

## 吹き出しの表示ルール

- 同じアイテムが同時に複数登場する場合、吹き出しは1つだけ表示されます。
- ストック済みのアイテムは吹き出しを出さず、マーカーのみ表示されます。
- アイテムごとに吹き出しを非表示にする設定もできます。
