# アイテムのタップ挙動・商品ポップアップ・投票
{: .no_toc }

紙面に配置したアイテムを閲覧者がタップしたときの挙動についての概要です。

## Tigアイコン（吹き出し）の表示

アイテムを置いた位置には吹き出し（Tigアイコン）が表示され、タップできる箇所であることを示します。閲覧者は設定から**吹き出しの表示／非表示を切り替え**られます。切り替えた状態は保持されます。

アイコンの種類・独自画像・大きさ・向き・背景色・文字色は管理画面から設定できます。アイテムごとに吹き出しを隠すこともできます。

## タップしたときの挙動

アイテムの種類と挙動設定の組み合わせで決まります。

| 種類 | タップしたときの挙動 |
|------|--------------------|
| 商品などのリンク（その場で遷移） | 別タブでリンク先を開きます。あわせてお気に入りに保存できます |
| 商品などのリンク（ポップアップ） | ポップアップの中にリンク先を表示します |
| 記事 | 画像・タイトル・本文の記事をポップアップで表示します |
| 投票 | 投票のポップアップを表示します |
| ページジャンプ | 指定したページへ移動します |
| 動画 | 動画をポップアップで再生します（紙面に直接埋め込む設定も可能です） |
| お買物メモを開く | お買物メモの一覧を開閉します |

リンク先のURLには、閲覧ページのパラメータやアイテムID、操作した時刻などを**差し込む書式**を利用できます。リンクの開き方（別タブ／同じタブなど）も指定できます。

## 商品ポップアップ

商品情報を整形して見せる専用のポップアップです。商品名・価格・商品説明・サムネイル・「商品詳細を見る」ボタンを表示します。

**利用の条件**は次のとおりで、いずれかが欠けると通常のポップアップ表示になります。

| 条件 | 内容 |
|------|------|
| コンテンツの設定 | 商品ポップアップを有効にしていること |
| アイテムの種類 | **商品アイテム**であること（管理画面で個別に作成したアイテムは対象外） |
| 閲覧環境 | **スマートフォン、または自社サイトへの埋め込み表示**であること（パソコンで直接閲覧した場合は対象外） |

商品説明が長い場合は「続きを読む」で展開できます。スマートフォンでは画面下部からせり上がる形で表示されます。

商品情報の取得に失敗した場合、ポップアップは表示されません（お気に入りへの保存とリンク先への遷移は行われます）。

## お買物メモ（ストック）

気になった商品をタップすると、お買物メモに保存されます。保存した内容は一覧から見返せます。投票アイテムも一覧に保存され、そこから投票画面を開き直せます。

## 投票

閲覧者が選択肢を選んで投票し、結果の割合を見られる機能です。

| 状況 | 表示 |
|------|------|
| 受付中・未投票 | 選択肢と送信ボタン |
| 投票したあと | 結果の割合（自分が選んだ選択肢が分かるように表示されます） |
| 受付終了後 | 結果の割合のみ |

- 投票したかどうかは**閲覧者のブラウザに記録**されます。同じ端末で開き直すと結果表示になります。
- **受付期限は締切を過ぎた時点で自動的に反映**されます。投票画面を開いたまま締切を迎えた場合も、その場で「受付終了」と結果表示に切り替わります。
- 結果の割合は 0% から実際の値までアニメーションで表示されます。最も多い選択肢は、受付終了後に色を変えて示します。
- 投票の受付期限そのものは画面に表示しません。
