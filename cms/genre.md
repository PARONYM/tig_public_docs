# ジャンル管理
{: .no_toc }

コンテンツを分類する「ジャンル」を管理する画面です。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## できること

| 操作 | 内容 |
|------|------|
| 一覧 | ジャンルを一覧表示。各ジャンルが使われているコンテンツ数も確認できます |
| 検索 | キーワードでジャンルを絞り込み |
| 追加 | 新しいジャンルを登録（登録前に同名の有無をチェックできます） |
| 編集 | ジャンル名を変更 |
| 削除 | ジャンルを削除 |

## 削除したときの動き

ジャンルを削除すると、そのジャンルが設定されていたコンテンツは「ジャンル未設定」に変わります（コンテンツ自体は削除されません）。
