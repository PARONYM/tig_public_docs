# タグ管理
{: .no_toc }

コンテンツや商品アイテムに付ける「タグ（分類ラベル）」を管理します。タグは専用画面ではなく、コンテンツ・アイテムの編集画面のタグ入力から追加・選択します。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## タグの種類

| 種類 | 内容 |
|------|------|
| グループのタグ | グループ単位で作成・利用するタグ。コンテンツやアイテムの分類・絞り込みに使います |
| 全体共通タグ（管理者用） | 管理者が作成する、全体で共通利用するタグ |

## 使い方

- コンテンツ・アイテムの編集画面で、タグを入力して追加したり、既存のタグから選んだりできます。
- グループのタグは、そのグループにアクセスできる権限のあるユーザーが追加できます。
- 作成したタグは、アナリティクスなどでの絞り込みにも利用できます。

> 流入元を区別する「計測タグ（ta_tag）」とは別のものです。
