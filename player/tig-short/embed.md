# Short の埋め込み・連携API
{: .no_toc }

Tig Short プレイヤーを外部サイトに埋め込む方法と、埋め込んだプレイヤーを外部サイトから制御・連携する API の概要です。

## 埋め込みについて

外部サイトに埋め込みコード（スクリプト）を設置すると、その場所に Tig Short プレイヤーが表示されます。1ページに複数の埋め込みを設置することもできます。埋め込み方法は用途に応じて複数用意されています（通常の埋め込み・SPAサイト向け・プレイリスト向けなど）。

## 外部サイトからの連携（API）

埋め込んだプレイヤーは、外部サイト側から操作したり、プレイヤーで起きたことを受け取ったりできます。

| できること | 内容 |
|-----------|------|
| 再生の制御 | 再生・停止・再生位置の取得/変更 |
| 表示の制御 | 再生ボタン・音量ボタン・ストックボタン・シェアボタン・チュートリアルなどの表示/非表示 |
| 通知の受信 | 視聴者がストックした・カートに追加したことの通知 |
