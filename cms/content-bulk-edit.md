# コンテンツ一括編集
{: .no_toc }

複数のコンテンツの内容を、CSVファイルを使ってまとめて編集する画面です。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 使い方の流れ

1. **CSVをダウンロード**します（Tig LIVE 用 / Tig Short 用のどちらかを選びます）。
2. ダウンロードしたCSVを表計算ソフトなどで開き、**タイトル・公開設定・（LIVEの）配信時間などを編集**します。
3. 編集したCSVを**アップロード**すると、変更した内容がまとめて反映されます。

## 編集できる内容

CSVの各項目（タイトル・公開設定・LIVEの配信時間など）を編集できます。変更があった行だけが更新されます。

## 反映されないとき・エラーになるとき

安全のため、次のような場合はアップロードが取り消され、**何も更新されません**（すべて元に戻ります）。

- 存在しないコンテンツIDが含まれている
- コンテンツIDが空、または重複している
- タイトルが空になっている
- 公開設定の値が正しくない
- LIVEの配信時間が不正、または変更できないタイミング（配信時間の変更は**配信開始の1時間前まで**、設定できるのは**現在から1時間以上先〜3か月未満**）
