# Tig Short コンテンツ一覧
{: .no_toc }

登録済みのコンテンツ（動画）を一覧・検索する画面です。各行から「詳細」を開いて編集します。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 一覧の見方

一覧は **リスト表示** と **ボックス表示** を切り替えられます。コンテンツ（サムネイル・タイトル）、コレクション、公開設定、登録日、更新日が確認できます。Instagram / TikTok から取り込んだコンテンツには UGC ラベルや SNS アイコンが付きます。

## 検索・絞り込み

キーワード、グループ、公開設定、コレクション、並べ替えで絞り込めます。設定した条件は画面を移動しても保持されます。

## 新規作成の3つの方法

- **新規アップロード**: 動画ファイルをアップロードして作成します。
- **動画を編集して作成**: 動画エディタで編集してそのまま作成します。
- **インポートして作成**: Instagram / TikTok の動画を取り込んで作成します（権利の確認を経て取り込みます）。
