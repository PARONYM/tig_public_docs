## 計測

視聴者の行動を計測し、アナリティクス画面で確認できます。

| 計測するもの | 例 |
|-------------|-----|
| 視聴 | 再生開始・視聴完了・離脱位置 |
| アイテム操作 | アイテムのタップ・ストック・リンク先への遷移 |
| 商品・クーポン | 商品詳細の表示・カート追加・クーポンの取得 |
| その他 | シェア・各種メニューの表示 |

- **視聴区間**（どこからどこまで見たか）は、シークやループなどの理由とともに記録され、途中でページを閉じても欠損なく送信されます。
- 内部確認用のプレビューや非公開プレビューでの視聴は**計測対象外**です。
- EEA圏では同意を得るまで計測しません（[読み込み・対応環境]({{ '/player/tig-video/load.html' | relative_url }})）。

## 外部サイトからの連携

外部サイトに埋め込んだプレイヤーは、サイト側から制御したり、視聴イベントを受け取ったりできます。

| できること | 内容 |
|-----------|------|
| 再生の制御 | 再生・停止・再生位置・音量・再生速度の操作 |
| 表示の制御 | ヘッダー・操作パネル・アイテム表示・ストック表示の切り替え |
| 通知の受信 | 再生開始・終了・アイテムのタップ・ストック・リンク遷移の通知 |

詳細な制御を行う API の利用には、発行されたキーが必要です。
