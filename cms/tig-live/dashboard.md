# Tig LIVE インサイト
{: .no_toc }

サイドバーの「インサイト」タブで、**配信が終了したライブ（直近20件）**の視聴指標をまとめて確認できる画面です。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 表示される内容

- **グラフ**: 各配信の LIVE 再生UU・アーカイブ再生UU（棒グラフ）、ストック率・ジャンプ率（折れ線）
- **コンテンツ情報**: サムネイル・配信日時・タイトル・紹介アイテム数（詳細ページへのリンク付き）
- **指標（LIVE／アーカイブ／通算 タブ）**: 総視聴数、視聴UU、ピーク視聴UU、平均視聴継続時間、コメント、Like、ストック、ジャンプ、視聴行動率、注目アイテム（ストック・ジャンプ上位）

> コンテンツ1件ごとの詳しい集計は[アナリティクス]({{ '/cms/tig-live/analytics-summary.html' | relative_url }})で確認できます。
