# プレイリスト（連続再生）
{: .no_toc }

複数の動画をまとめて連続で視聴でき、カテゴリやフィルタで見たい動画に絞り込める「プレイリスト」の説明です。

## プレイリストとは

複数の Short 動画をひとつのまとまりとして連続再生できる機能です。プレイリストが設定されたコンテンツでは、視聴中に次々と動画が切り替わります。

## できること

| 操作 | 内容 |
|------|------|
| 連続再生 | プレイリストに含まれる動画を続けて視聴できます |
| スワイプで切り替え | 上下のスワイプで前後の動画に移動できます（初回はスワイプの案内が表示されます） |
| カテゴリで絞り込み | 「すべて」または特定のカテゴリを選ぶと、その内容に切り替わります |
| フィルタで絞り込み | フィルタ条件を選ぶと、条件に合う動画だけに絞り込めます |
| バナー・詳細 | プレイリストのバナーやリンクから、関連情報を確認できます |

## 外部サイトへの掲載

プレイリストは、外部サイトに**サムネイル付きの再生ボタン**として埋め込むこともできます。ボタンをクリックすると、その場でプレイリストがモーダル（ポップアップ）で開いて再生できます。
