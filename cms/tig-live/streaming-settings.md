# Tig LIVE 配信設定（全体設定）
{: .no_toc }

サイドバーの「全体設定」から開く、グループ全体に関わる設定です。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 全体設定の構成

「全体設定」には2つのメニューがあります。

| メニュー | 内容 |
|---------|------|
| テンプレート管理 | 配信の[配信テンプレート]({{ '/glossary.html' | relative_url }})を管理する |
| 配信設定 | 入力映像・カメラ解像度・ビットレートなどのグループ共通の配信設定 |

## 配信設定

「配信設定」では、グループ共通の配信に関する設定ができます。

| 項目 | 内容 |
|------|------|
| 入力映像サイズ | 「Tig LIVE Mixer（標準）」または「カスタム」（カスタム時は横×縦を指定） |
| カメラキャプチャ解像度 | HD / フルHD（FHD） |
| 映像出力ビットレート | 1 / 1.5 / 2 / 3 / 5 / 10 Mbps から選択 |

> コンテンツごとの映像設定は、各配信の[配信詳細]({{ '/cms/tig-live/content-detail.html' | relative_url }})の「映像・配信」からも調整できます。
