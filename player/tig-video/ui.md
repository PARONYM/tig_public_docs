# シェア・再生前後の画面
{: .no_toc }

再生前のサムネイル表示、動画末尾の画面、シェア機能などの概要です。

## 再生前の画面（フライヤー）

再生前はサムネイル画像と再生ボタンが表示されます。自動再生を設定している場合は表示されませんが、**ブラウザ側で自動再生が制限された場合は再びサムネイルが表示**され、タップで再生できます。

## 動画の最後の画面

動画を最後まで再生したときの画面は、コンテンツごとに選べます。

| 種類 | 内容 |
|------|------|
| シェア | SNSでの共有・リンクのコピー |
| ストック | ストックしたアイテムの一覧 |
| リプレイ | もう一度再生する導線 |

## シェア

Facebook・X・メール・リンクのコピーで共有できます。スマートフォンでは端末標準の共有メニューも利用できます。

## チャプター（分岐動画）

分岐のある動画では、チャプター一覧から任意の章や最初の画面へ移動できます。

## 表示言語・配色

- UIの文言は日本語／英語に対応し、視聴環境の言語設定に応じて切り替わります。
- シークバーの色やアイテムのマーカー色などは、コンテンツごとの設定で変更できます。動画の余白（レターボックス）の色や、吹き出しのフォントも設定できます。
