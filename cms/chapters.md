# チャプター（目次）
{: .no_toc }

コンテンツにチャプター（目次の区切り）を設定する機能です。コンテンツ編集画面の中で設定します。

{: .no_toc .text-delta }

1. TOC

## できること

| 操作 | 内容 |
|------|------|
| 追加 | タイトル・説明・サムネイル画像を指定してチャプターを追加 |
| 編集 | チャプターの内容やリンク先を変更 |
| 画像の変更 | チャプターのサムネイル画像を差し替え |
| 削除 | チャプターを削除 |

## 使われる場面

チャプター（目次）は、次のコンテンツ編集で使われます。

- **Tig Branch Video**: 分岐チャプター（別のコンテンツや再生位置へのリンク）
- **Tig Magazine**: 目次
- **Tig LIVE のアーカイブ**: チャプター

各チャプターには、別コンテンツ・再生位置へのリンクを設定できます。
