# ウェルカムページ
{: .no_toc }

Tig の利用を始めるにあたって必要な設定や、直近の実績・お知らせをまとめて確認できるホーム画面です。対象のグループでは、ログインするとこのページが最初に表示されます。

## 表示される方

このページは、EC連携の導入ガイドが有効になっているグループ（LIVE または Short をご利用の場合）と、管理者の方に表示されます。表示されない場合も、通常の各画面はそのままご利用いただけます。

## 画面の構成

| セクション | 内容 |
|-----------|------|
| ECサイト導入までのタスク | 利用開始までに必要な設定（ドメイン登録・計測の設定など）の進み具合をチェックリストで確認できます。「設定」リンクから各設定画面へ移動できます |
| Tig コマース実績 (前回) | 前日の実績（視聴や購入などのサマリー）を確認できます |
| 利用可能なサービス | ご契約中のサービス（LIVE / Short など）と、その管理画面への導線が表示されます |
| Tig インフォメーション | Tig からのお知らせが表示されます |
| オンボーディング | LIVE / Short それぞれの利用開始ステップの進み具合と、使い方を案内する動画を確認できます |

## 使い方のヒント

- まずは「ECサイト導入までのタスク」を上から順に完了させると、計測や配信の準備が整います。
- タスクの完了状況は設定内容に応じて自動で更新されます。
