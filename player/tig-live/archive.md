## アーカイブ視聴について

配信終了後に公開されたアーカイブ（録画）を、LIVE視聴と同じ操作感で再生できます。動画上には配信中に紹介されたアイテム（TIGポイント）が同じタイミングで表示され、タップして商品情報の確認やストックができます。

## 画面の機能

| 機能 | 内容 |
|------|------|
| 出演者情報 | 出演者の一覧・プロフィール・関連リンクを表示 |
| アーカイブ一覧 | 同じ配信グループの他のアーカイブや次回配信の案内を表示 |
| お知らせ | カレンダー登録・LINE友だち追加・メルマガ購読などの導線 |
| 紹介アイテムの履歴 | それまでに紹介されたアイテムを一覧表示。タップすると紹介された場面へジャンプ |
| 前後のアイテムへスキップ | アイテムが紹介された場面単位で前後にスキップ |
| ヘルプ | 操作方法の案内画像を表示 |
| AIコーディネート | ストックしたアイテムに関連するおすすめ商品の一覧を表示（設定時） |

## 操作について

- 再生・停止・音量・全画面・シークバーなどの基本操作は動画プレイヤーと共通です。
- プレイヤーの配色（シークバーの色など）は、コンテンツごとの設定に応じて変わります。
- 動画の最後まで再生すると、次のアクション（他のアーカイブや案内）への導線が表示されます。
