# ユーザー管理
{: .no_toc }

CMS を利用するユーザー（アカウント）を管理する画面です。ヘッダーの「その他管理」から開きます。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## できること

| 操作 | 内容 |
|------|------|
| 一覧・検索 | ユーザーを一覧表示。所属グループ・キーワードで絞り込みできます |
| 登録 | メール・名前・所属グループ・ロール（権限）を指定して新規追加 |
| 編集 | 既存ユーザーの情報を変更 |
| 削除 | ユーザーを削除（過去データは保持されます） |
| 仮パスワード発行 | 登録時に、ログイン用の仮パスワードを発行できます |

## 表示・操作できる範囲

ユーザーの一覧・登録・編集・削除は、**自分が管理できるグループの範囲**に限られます（権限の設定によります）。グループ単位までの権限の場合、登録・編集時の所属グループは自分のグループに固定されます。

## ロール（権限）

ユーザーには[ロール（権限）](roles.html)を割り当てます。割り当てられるロールは、所属グループの種類によって変わります。

## ログイン

CMS へのログイン画面もこの機能の一部です。発行された仮パスワードでログインできます。
