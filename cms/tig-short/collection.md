# コレクション
{: .no_toc }

Tig Short のコンテンツを「コレクション」としてまとめる画面です。コレクション単位でまとめて見せたり、絞り込みやフローティング表示に使えます。

## できること

| 操作 | 内容 |
|------|------|
| 一覧 | コレクションを一覧表示（グループで絞り込み） |
| 新規作成 | 名前などを指定してコレクションを作成 |
| コンテンツの追加・除外 | コレクションに含めるコンテンツを選ぶ |
| 編集 | 名称・OGP画像などを変更 |
| 並び替え | コンテンツの並び順（登録順／自由な順）を変更 |
| プレビュー | 表示イメージを確認 |
| 削除 | コレクションを削除 |

## 補足

- コレクションには公開用のURLや埋め込みコード、OGP画像を設定できます。
- コレクションは[プレイリスト](playlist.html)にまとめて使うこともできます。
- 保存すると、プレーヤー側の表示は自動で最新化されます。
