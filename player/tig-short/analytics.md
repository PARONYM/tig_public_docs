# 計測（アナリティクス）
{: .no_toc }

プレイヤーでの視聴・操作を計測し、CMS のアナリティクスや EC ダッシュボードに反映するしくみの説明です。

## 何を計測しているか

プレイヤーは、視聴者の行動を集計用に計測しています。取得したデータは CMS の分析画面（アナリティクス・EC ダッシュボード）で確認できます。

| 種類 | 内容 |
|------|------|
| 再生 | 動画の再生開始・再生完了 |
| 視聴範囲 | 動画のどこをどれだけ見たか（再生完了率の算出に使用） |
| タップ | 動画や商品ポイントのタップ |
| ストック | 商品をストックした操作 |
| リンク遷移 | 商品ページなどURLへの遷移 |
| ポップアップ | 商品モーダルの表示 |
| カート追加 | EC商品のカート追加 |
| 操作 | 前後スキップなどの操作 |

## プライバシーについて

計測は匿名の識別子（利用者を特定しないID）で行っており、個人を特定する情報は取得しません。
