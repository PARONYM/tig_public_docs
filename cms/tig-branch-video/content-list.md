# Tig Branch Video コンテンツ一覧
{: .no_toc }

Tig Branch Video（複数の動画を分岐でつなぐインタラクティブ動画）のコンテンツを一覧で管理する画面です。検索・絞り込みのほか、新規作成・複製・削除ができます。

## 画面の構成

この画面は大きく2つの領域に分かれています。

- **ヘッダー部分（画面上部）**: キーワード検索・全件表示、**新規作成**ボタン、件数表示、絞り込み（グループ〔代理店以上〕・公開設定）、表示の切り替え（ボックス／リスト）、並べ替え。
- **メイン領域（画面下部）**: Branch Video の一覧。各行にサムネイル・タイトル・公開状態・状態（コピー中）・登録日／更新日が表示され、行から詳細画面へ進めます。

以降は、各操作を説明します。

## 使い方

1. キーワードや絞り込み（グループ・公開設定）で目的のコンテンツを探す
2. 表示形式（ボックス／リスト）や並べ替えを切り替える
3. 行をクリックして詳細画面で動画ノード・分岐を編集する
4. 新しく作る場合は「新規作成」から作成する

## 新規作成

「新規作成」では、まず**タイトルだけ**を入力して空の Branch Video を作成します。動画の追加や分岐のつなぎ込みは、作成後の詳細画面で行います。

## 複製・削除

- **複製**: 動画ノードや分岐を含めて Branch Video 全体を複製できます（複製中は「コピー中」と表示）。
- **削除**: Branch Video を削除します（一覧から消えます）。

## ご利用の前提

- Tig Branch Video は**分岐動画機能が有効なグループ**でのみ利用できます。
