# 外部サイトへの埋め込み

## 表示形式

| 形式 | 内容 |
|------|------|
| フローティング（既定） | ページ上に小窓を固定表示。プレビュー動画が流れ、タップで本編が開く |
| ページ内埋め込み | 指定位置に表示し、スクロールで画面外に出ると隅に追従（PiP） |
| 全画面 | ページ全体にプレイヤーを表示 |
| クリックモーダル | ページ内の要素をクリックしてポップアップ表示 |

## フローティングの設定

- プレビュー動画の上に**説明テキスト（複数行可）**やバナーを表示できます。
- 小窓の閉じ方は「**最小化**（アイコンになり、再表示できる）」と「**閉じる**（そのページでは再表示しない）」から選べます。

## ページ内埋め込み・追従表示

ページ内に埋め込んだプレイヤーは、スクロールで画面外に出ると画面の隅に小さく追従し、元の位置に戻ると復帰します。追従中の閉じるボタンは、最小化・完全に閉じるのいずれかを設定できます。

## 全画面表示

ページ全体でプレイヤーを表示します。埋め込み元ページのURLパラメータは、そのままプレイヤーへ引き継がれます。

## 計測

埋め込みの表示・再生・クローズは計測され、お客様サイトの解析ツールやコンバージョン計測タグにも連携できます。

埋め込み全体の考え方は[埋め込み連携の全体像]({{ '/player-common/embed-overview.html' | relative_url }})をご覧ください。
