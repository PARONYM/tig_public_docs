# アップローダーライブラリ
{: .no_toc }

音声・画像・動画などのファイルをアップロードして保管し、配信用のURLで再利用するための画面です。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 画面の構成

- **メイン領域**: アップロード済みのファイルが一覧で表示されます（グループごとに管理されます）。各ファイルの配信用URLを確認できます。
- **アップロード操作**: ファイルを選んでアップロードすると、一覧に追加されます。

## できること

| 操作 | 内容 |
|------|------|
| 一覧 | アップロード済みのファイルを一覧表示（100件ずつ表示・並び替え可）。各ファイルの配信用URLを確認できます |
| アップロード | ファイルをアップロードして保管します |

## アップロードできるファイル

次の形式に対応しています。

- 画像: PNG / JPEG
- 音声: MP3
- 動画: MP4

安全のため、アップロード時にファイルの検査を行い、対応形式以外や問題のあるファイルは受け付けません。
