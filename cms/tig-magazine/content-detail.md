# Tig Magazine コンテンツ詳細
{: .no_toc }

Tig Magazine コンテンツの設定・編集を行う画面です。タブを切り替えて、ページ・目次・アイテム・公開設定などを編集します。

## 画面の構成

この画面は大きく2つの領域に分かれています。

- **ヘッダー部分（画面上部）**: サムネイル、動画（ページ）の差し替え、プレビュー、複製・削除、公開URL、ヒートマップ／アナリティクスへの導線、状態表示（変換中・コピー中）。
- **メイン領域（画面下部）**: タブ（基本設定・ページ管理・目次設定・Tig Video設定・アイテム設定・Tigアイコン設定・公開設定）を切り替えて各種設定を編集します。

以降は、各タブでできることを説明します。

## メイン領域での操作（タブ）

| タブ | 内容 |
|------|------|
| 基本設定 | タイトル・説明・SEO・ジャンル・所有グループ・マガジンの表示設定（見開き・表紙・めくり方向／アニメーション など） |
| ページ管理 | ページ画像の追加・並び替え・削除・差し替え |
| 目次設定 | 目次の表示と、各目次のページ指定 |
| Tig Video設定 | マガジンのページに Tig 動画を埋め込み、再生オプションを設定（対応グループのみ） |
| アイテム設定 | ページに表示するアイテム（個別アイテム・商品アイテム）と、**投票アイテム**の追加・編集 |
| Tigアイコン設定 | アイコンの種類・画像・サイズ・色・不透明度・省略表示 |
| 公開設定 | 公開モード（公開／非公開／期間指定）・視聴パスワード・IP／ドメイン制限 |

## ページ管理

マガジンはページ画像の集まりです。ページ管理タブから、ページの**追加・並び替え・削除・差し替え**ができます。並び替えると、目次やアイテムの配置も一緒に整います。

## アイテム・投票

アイテム設定タブでは、ページに表示するアイテム（個別アイテム・[商品アイテム]({{ '/cms/item-master.html' | relative_url }})）を設定できます。**投票アイテム**（複数の選択肢から選んでもらうアンケート）も追加でき、期限を設定できます。

## プレビュー・公開

- **プレビュー**: 編集中の見え方をモーダルで確認できます。
- **公開設定**: 公開／非公開／期間指定を切り替え、視聴パスワードやアクセス制限も設定できます。
