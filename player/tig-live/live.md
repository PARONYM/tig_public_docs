# プレイヤー（LIVE）
{: .no_toc }

LIVE配信をリアルタイムで視聴するプレイヤーの概要です。

## 視聴の流れ

| 段階 | 画面 |
|------|------|
| 配信前 | 待機画面（フィラー）が表示されます。開始までのカウントダウンや、配信者情報・お知らせ・過去の配信一覧を確認できます |
| 配信中 | ライブ映像が再生され、紹介タイミングに合わせて商品などのアイテム（TIGポイント）が表示されます。タップして詳細確認やストックができます |
| 配信終了後 | 終了画面（フィラー）に切り替わり、配信中に紹介されたアイテムの一覧・ストック・購入導線・アンケートを利用できます |

配信の開始・終了は自動で検知され、視聴者の操作なしで画面が切り替わります。指定時刻に自動で始まる配信（プレミア公開）にも対応しています。

## 配信中の機能

| 機能 | 内容 |
|------|------|
| アイテム表示 | 配信のタイミングに同期して商品・クーポン・記事などが表示される |
| 紹介アイテムの履歴 | それまでに紹介されたアイテムを一覧から確認・ストック |
| 配信者情報 | 出演者のプロフィール・関連リンクを表示 |
| 投票 | 配信中の投票に参加でき、結果がリアルタイムに反映される |
| AIコーディネート | 紹介アイテムに関連するおすすめ商品の一覧を表示（設定時） |

コメント・いいねなどの参加機能は別ページで説明します（準備中）。

## 配信終了後の機能

- 紹介アイテムの一覧・ストック・商品購入（EC連携）への導線が表示されます。
- 配信終了から24時間以内はアンケートに回答できます（設定時）。
- アーカイブ（録画）が公開されると、[プレイヤー（アーカイブ）]({{ '/player/tig-live/archive.html' | relative_url }})で視聴できます。

## LINE での視聴

LINE アプリ内での視聴に対応した配信では、LINE アカウントにひも付いた形でストックなどが保存されます。
