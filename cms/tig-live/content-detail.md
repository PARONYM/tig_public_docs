# Tig LIVE 配信詳細
{: .no_toc }

ライブ配信の設定・配信操作を行う画面です。左のサイドメニューから各設定タブを切り替えます。表示されるタブは配信の状態（配信前／配信中／配信終了／アーカイブ）に応じて変わります。

## 画面の構成

- **サイドメニュー（画面左）**: サムネイル・配信ステータスと、設定タブ（ライブ／画像／アイテム／コンテンツ／映像配信／埋め込み／オプション など）。配信の状態に応じて表示されるタブが変わります。
- **メイン領域（画面右）**: 選んだタブの設定内容を編集します。
- **画面上部**: 公開モードの切り替え、アナリティクス／ヒートマップへの導線。

以降は、主な設定でできることを説明します。

## 使い方

1. [配信一覧]({{ '/cms/tig-live/content-list.html' | relative_url }})で対象のコンテンツを開く
2. 左のサイドメニューで設定タブ（ライブ／画像／アイテム／コンテンツ／映像配信／埋め込み／オプション）を選ぶ
3. 各タブで設定し「保存」する。配信後の集計は[アナリティクス]({{ '/cms/tig-live/analytics-summary.html' | relative_url }})で確認できます

## 公開モード

コンテンツの公開状態を設定します。

| モード | 内容 |
|--------|------|
| 公開 | 視聴者が視聴できる状態 |
| 非公開 | 視聴者から見えない状態 |
| 期間指定 | 公開する開始・終了の日時を指定 |

> 公開するには、タイトル・OGP画像・フィラー画像の設定が必要です。

## 基本設定

タイトル・説明・配信開始日時・配信時間・アイテムの表示位置・スタッフ用パスワード・SEO設定・自動アーカイブなどを設定します。

## 配信操作

- **ライブの停止（途中終了）**: 予定時間より前にライブを終了できます。
- **アーカイブの作成・公開**: 配信終了後に、録画をアーカイブとして作成・公開できます。
- **削除**: コンテンツを削除します。

## 自動アーカイブ・自動公開

基本設定で、配信終了後のアーカイブの扱いを次の3つから選べます。

| 設定 | 動作 |
|------|------|
| 自動で作成しない（初期値） | アーカイブは手動で作成します |
| 配信終了後に自動で作成 | 配信終了後、アーカイブが自動で作成されます（公開は手動） |
| 配信終了後に自動で作成して公開 | アーカイブの作成完了後、自動で公開まで行います |

- 自動作成は配信終了後に順に処理され、**公開までの目安は最長で1時間程度**です。
- アーカイブを作り直した場合は、意図せず再公開されないよう、設定を「自動で作成しない」に戻して保存してください。

## 画像

サムネイル・OGP画像・フィラー画像・ロゴ画像を設定します。フィラー画像は「事前」「オープニング」「ブレイク」「エンド」「事後」の場面ごとに設定でき、まとめて一括設定もできます。

## 共有

視聴者向けの**視聴URL**と、関係者向けの**スタッフモードURL／パスワード**を確認・コピーできます。QRコードも表示されます。

## アイテム

コンテンツに表示するアイテムを設定します。

| 種類 | 内容 |
|------|------|
| 商品アイテム | 紹介する商品（[商品アイテム管理]({{ '/cms/item-master.html' | relative_url }})のマスターから設定） |
| 配信者 | 配信者の情報・リンク |
| クーポン | クーポンコード |
| 投票・アンケート | 視聴者参加の投票／アンケート |
| 個別アイテム | このコンテンツ専用のアイテム（一括登録も可能） |
| URLパラメータ | アイテムのリンクに付与する計測用パラメータ |

## 映像配信

配信の画質（HD／FHD）・入力サイズ・配信品質を設定します。あわせて、**サイマル配信**（YouTube／Instagram／TikTok／X への同時配信）の送信先を設定できます。

## 埋め込みコード

自社サイトにプレーヤーを設置するための埋め込みコードを発行します。ダイレクト／モーダル／フローティングの形式から選べ、表示オプションを設定してコピーできます。

## コンテンツ関連付け

プレーヤー上の視聴者向け導線を設定します。LINE友だち追加・メールマガジン登録のURL、カレンダー連携、アーカイブ一覧の表示などを設定できます。

## オプション

| 設定 | 内容 |
|------|------|
| コンテンツ管理 | タグ・説明・NGワードの管理 |
| 公開制限 | IPアドレス・ドメイン・パスワードによる視聴制限 |
| 動画管理 | アーカイブの作成・削除・コンテンツ化・ダウンロード |
| 字幕 | 自動字幕の生成・編集（日本語／英語／スペイン語／中国語／韓国語） |

## プレミア公開

事前にアップロードした動画を、指定した時刻に**自動でライブ開始**する公開方法です。あらかじめ収録した映像を、ライブのように定刻配信できます。
