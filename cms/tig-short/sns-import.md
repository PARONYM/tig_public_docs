# SNS/外部動画の取り込み
{: .no_toc }

Instagram／TikTok の動画を取り込んで Tig Short のコンテンツを作成する機能です。コンテンツ作成画面から利用します。

## 目次
{: .no_toc .text-delta }

1. TOC
{:toc}

## 使い方

1. 事前に[SNSログイン連携設定](../sns-login.html)で Instagram／TikTok を連携します。
2. 連携したアカウントの動画一覧を表示し、取り込む動画を選びます。
3. 取り込みを実行すると、動画の取得処理が進みます（進捗が表示されます）。
4. 取り込んだ動画から Tig Short のコンテンツを作成できます。
5. 取り込んだ動画は削除もできます。

## 補足

- 取り込みには時間がかかる場合があり、完了状況は画面で確認できます。取り込めなかった動画がある場合はお知らせします。
- モバイルアプリからも取り込みできます。
- LIVE のアーカイブとして取り込む場合は[配信の新規作成](../tig-live/content-create.html)から行います。
